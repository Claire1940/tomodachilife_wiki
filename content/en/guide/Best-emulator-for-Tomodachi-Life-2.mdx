## Troubleshooting: Crashes, Black Screen, and Low FPS

Even with the **Best emulator for Tomodachi Life 2**, issues can happen after updates. Use this fast triage flow instead of random changes.

### Common issues and fixes

| Problem | Likely Cause | Fast Fix |
|---|---|---|
| Game crashes on boot | Wrong emulator branch or outdated files | Update Ryujinx build; verify firmware/keys |
| Black screen after launch | Graphics backend mismatch | Switch Vulkan ↔ OpenGL test |
| Big frame drops | Resolution too high for GPU | Drop to native 1x, reduce AA/filtering |
| Input not detected | Incorrect profile/device mapping | Refresh devices, reload saved profile |
| Random hitching | Shader cache building | Play longer in same scenes; avoid constant setting changes |

### Minimal-change troubleshooting order

1. Confirm you’re on a current build.
2. Validate firmware and keys installation.
3. Test default settings at 1x resolution.
4. Confirm dedicated GPU is selected.
5. Re-enable enhancements one by one.

This approach helps you preserve performance while isolating the real fault. It’s the most reliable way to maintain the **Best emulator for Tomodachi Life 2** setup over time.
