<Section icon={Target} title="Step-by-Step Setup Guide">

Setting up **tomodachi life keyboard controls** using the RMB script method is the most reliable way to ensure your mouse and keyboard work in harmony. Follow these steps to configure your environment for the "Living the Dream" experience.

<Steps>
  <Step num="1" title="Download and Extract Scripts">
    Locate the "RMB Windows" or "RMB Writings" folder. Right-click the compressed file and select **Extract All** to access the configuration files.
  </Step>
  <Step num="2" title="Launch the Control Utility">
    Open the extracted folder and run the executable file (usually named `RMB_Writings.exe`). This utility bridges the gap between your mouse movements and the emulator's touch input.
  </Step>
  <Step num="3" title="Initialize the Game">
    Start your emulator and load Tomodachi Life. Once the title screen appears, click **Start** within the RMB utility to sync the inputs.
  </Step>
  <Step num="4" title="Configure In-Game Settings">
    Go to the emulator's Input Settings. Ensure "Enable Mouse" is checked and set the "Stick" sensitivity to 100% to prevent lag during fast-paced movements.
  </Step>
  <Step num="5" title="Test and Calibrate">
    Enter the Mii Apartments and try dragging a Mii's head with the mouse. If the movement is jittery, toggle the **Hide Mouse** option in the utility settings.
  </Step>
</Steps>

<Callout type="success" title="Success">
Once configured, you can use **Control + F9** at any time to safely close the game and the script utility simultaneously.

<Section icon={MousePointer} title="Mastering Mouse and Touch Interactions">

Since Tomodachi Life was designed for a stylus, the mouse is actually the most important part of your **tomodachi life keyboard controls**. Many players struggle with the "swiping" motion required for activities like patting a Mii's head or playing the "Tickle" game.

| Mouse Action | Game Result | Usage Scenario |
| :--- | :--- | :--- |
| **Left Click** | Stylus Tap | Selecting items, talking to Miis |
| **Left Click + Drag** | Stylus Swipe | Patting Miis, throwing items, sliding menus |
| **Right Click** | Context Menu | (In some scripts) Opens inventory quickly |
| **Scroll Wheel** | Zoom In/Out | Viewing the island or room interiors |

Using a high-DPI mouse is recommended for the "Catch" mini-game, where you must react instantly to a falling object. If your cursor feels "heavy," check if your emulator is using **V-Sync**, which can introduce input latency.

<Callout type="info" title="Technical Note">
The RMB Writings script allows you to toggle the mouse cursor's visibility. Hiding the cursor often improves immersion, as the in-game hand icon will follow your invisible mouse movement.
</Callout>
