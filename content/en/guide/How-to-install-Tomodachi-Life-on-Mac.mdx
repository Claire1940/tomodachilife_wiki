## Troubleshooting: Common Mac Issues and Fixes

If something breaks after you follow **How to install Tomodachi Life on Mac**, use this quick diagnostic flow.

### Game won’t boot

- Confirm file integrity (bad dump = black screen/crash)
- Update emulator to latest stable build
- Reset graphics backend to default
- Remove custom cheats/mod patches and retest

### Heavy stutter every few seconds

- Expected during first shader compilation
- Let game run in same area for a few minutes
- Ensure power mode is not low-power
- Lower resolution one step

### No sound or distorted audio

- Switch output device in macOS sound settings
- Toggle audio stretching
- Restart emulator after changing backend

### Controls not detected

- Re-pair Bluetooth controller
- Disable Steam Input or other remappers temporarily
- Rebind controls inside emulator profile
