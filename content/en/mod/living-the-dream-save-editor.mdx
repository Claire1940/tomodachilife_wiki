## Preparing for Modification: File Locations

Before you can use the **living the dream save editor**, you must be able to locate and backup your save files. Depending on whether you are using the [Ryujinx emulator](https://ryujinx.org/) or a physical Nintendo Switch with custom firmware, the process for finding these files differs slightly.

### Finding Save Data on Ryujinx

For most players on PC, Ryujinx is the primary method for experiencing this project. The emulator stores save data in a nested directory within your system's AppData folder.

1.  Press `Windows Key + R` and type `%appdata%`.
2.  Navigate to `Ryujinx` > `bis` (or `bin`) > `user` > `save`.
3.  You will see several folders with long numerical strings. You must look through these until you find the one containing `player.save`.
4.  **Important:** Always create a backup copy of your `player.save` before proceeding with any edits.

### Finding Save Data on a Modded Switch

If you are playing on hardware, you will need a save manager such as JKSV or Checkpoint.

*   Export the save data for "Tomodachi Life: Living the Dream" using your preferred manager.
*   Connect your SD card to your PC or use an FTP client to access the exported folder.
*   Look for the `player.save` file within the exported directory.

> **Warning:** Modifying save files on a physical console carries a small risk of data corruption. Ensure your console is in Airplane Mode or blocked from official servers to prevent any synchronization issues.

## Troubleshooting Common Issues

While the **living the dream save editor** is highly reliable, users may encounter minor hurdles when first attempting to mod their game.

### Save File Not Loading
If Ryujinx fails to boot after you replace the file, you may have renamed the file accidentally. Ensure the filename is exactly `player.save` (all lowercase) and that it is placed in the correct numerical subfolder.

### Miis Still Mentioning the Retail Version
This occurs if the "Lockdown Status" wasn't correctly toggled or if you didn't overwrite the correct file. Re-run the editor and verify that "Status Lockdown" is set to "Disabled" before downloading the new save.

### Happiness and Leveling
In the "Welcome Version," the developers may have capped the happiness bar. While the **living the dream save editor** allows Miis to interact, they may not always gain levels or "Warm Fuzzies" in the same way they would in the full retail version. This is an engine limitation of the demo itself rather than a fault of the editor.

## FAQ

**Q: Can I use the living the dream save editor on the original 3DS version of Tomodachi Life?**

A: No, this specific editor is designed for the "Living the Dream" fan project and demo for the Nintendo Switch/Ryujinx. For the 3DS version, you would need a different tool specifically made for `.sav` files from that hardware.

**Q: Is it possible to add more than three Miis using this tool?**

A: The current version of the editor focuses on player stats, money, and items. While disabling the lockdown allows your existing Miis to move freely, the ability to add more Miis is often limited by the demo's internal code. However, some users have had success by creating Miis before the lockdown triggers.

**Q: Will using the living the dream save editor get my Nintendo account banned?**

A: If you are using an emulator like Ryujinx, there is no risk to your official Nintendo account. If you are using a modded Switch, you should always stay offline when using modified save data to avoid detection by Nintendo’s telemetry services.

**Q: Does the editor work on Mac or Linux?**

A: Yes. Since the editor is a web-based tool provided by the community, it functions in any modern browser regardless of your operating system. You only need to be able to access your file system to upload and download the `player.save` file.
