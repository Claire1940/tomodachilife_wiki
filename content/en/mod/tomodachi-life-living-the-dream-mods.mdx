## Installation Guide for Mods and Patches

Installing **tomodachi life living the dream mods** varies depending on whether you are using hardware like the Nintendo Switch or software like the Ryujinx emulator. Most mods are distributed as either save files (`.save`) or Atmosphere patches (`.ips`).

### For Emulator Users (Ryujinx)
1. Right-click the game in your library and select "Open User Save Directory."
2. Navigate to the numbered folder (usually `0`) to find your `player.save`.
3. Use a web-based save editor to modify this file, then overwrite the original.
4. For graphical mods, place the `.ips` files in the `atmosphere/exefs_patches/` directory within your emulator's virtual SD card.

### For Modded Switch Users (Atmosphere)
1. Connect your SD card to your PC or use an FTP client.
2. Place content mods in `sd:/atmosphere/contents/[GameID]/romfs/`.
3. Place code patches (like graphics mods) in `sd:/atmosphere/exefs_patches/[ModName]/`.
4. Ensure the `.ips` file is named correctly according to the game's build ID.

## FAQ

**Q: Are tomodachi life living the dream mods safe to use on my console?**

A: Generally, yes. Cosmetic and save-based mods carry very little risk. However, using mods while connected to official Nintendo servers can sometimes lead to account restrictions. It is always recommended to use an "emuNAND" or stay offline when playing with modified files.

**Q: Why are my Miis still stuck in their houses after applying the save editor?**

A: Ensure that you correctly selected "Disable Lockdown" in the editor and successfully overwrote the `player.save` file in the correct directory. If you are using an emulator, make sure you are editing the save for the specific version (Demo vs. Full) that you are currently launching.

**Q: Can I use these mods on a non-modded Nintendo Switch?**

A: No. These mods require access to the system's file structure, which is only possible through custom firmware (CFW) like Atmosphere or by using a PC-based emulator.

**Q: Where can I find the Game ID for Tomodachi Life: Living the Dream?**

A: The Game ID can usually be found by right-clicking the game in your emulator and selecting "Properties" or by checking the title ID list on your CFW's homebrew menu. It is essential for placing mods in the correct `atmosphere/contents/` folder.
