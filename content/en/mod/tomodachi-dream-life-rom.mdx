## How to Configure Your Tomodachi Dream Life ROM on Ryujinx

1.  **Install Keys:** Open Ryujinx, navigate to *File* > *Open Ryujinx Folder*. Place your `prod.keys` in the `system` folder. Alternatively, use the *Actions* menu to install keys directly.
2.  **Install Firmware:** Go to *Tools* > *Install Firmware* and select your firmware ZIP or XCI file.
3.  **Add Game Directory:** Go to *Options* > *Settings*. Under the "General" tab, click *Add* and point the emulator to the folder where your **tomodachi dream life rom** is stored.
4.  **Configure Graphics:** Navigate to the *Graphics* tab. It is highly recommended to set the **Graphics Backend** to **Vulkan**.

## Customizing Textures with TomoTexture

One of the most exciting aspects of the **tomodachi dream life rom** experience is the ability to use **TomoTexture**. This "magical" utility allows players to export and replace in-game textures, such as food items, clothing patterns, and even Mii face paints.

### Setting Up the User Save Directory
To use TomoTexture, the program needs to know where your save data is located. 

*   Right-click on Tomodachi Life in the Ryujinx game list.
*   Select *Open User Save Directory*.
*   Copy the folder path (usually ending in a string of zeros).
*   Paste this path into the TomoTexture "Save Directory" field and hit refresh.

### Replacing Textures
When you decide to replace a texture (for example, turning a standard cake into a custom "glazed cake"), the program will prompt you to select a backup folder. This is a safety feature. By creating a dedicated `UGC Backups` folder, you ensure that if a texture edit goes wrong, you can revert the **tomodachi dream life rom** to its original state without reinstalling the entire game.

> **💡 Tip:** If you are editing face paints, use a transparent PNG format. This allows the Mii's natural skin tone to show through the custom design, creating a more seamless look.
