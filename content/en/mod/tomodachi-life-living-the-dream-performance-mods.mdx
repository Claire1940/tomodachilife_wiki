## Crash Fixes and “Game Not Showing” Troubleshooting

Most complaints in 2026 still fall into two buckets: immediate crash on launch, or game not appearing in library. Use this matrix before changing advanced settings.

| Symptom | Likely Cause | Fix Priority | Practical Fix |
|---|---|---|---|
| **Crash on boot** | Outdated GPU driver | High | Update driver, reboot, retest |
| **Crash after enabling mod** | Wrong mod version/structure | High | Disable mod, verify folder depth and version |
| **Game missing from list** | Wrong game root path | High | Re-add correct parent folder and refresh |
| **Random stutter spikes** | Shader compilation/load spikes | Medium | Let shaders build naturally, avoid rapid setting flips |
| **Periodic instability** | Too many active mods | Medium | Strip to FPS patch only, then reintroduce one by one |

### Fast diagnostic routine (5-minute pass)

1. Disable all mods.
2. Confirm game launches clean.
3. Re-enable only 60 FPS mod.
4. Test one session (10 minutes).
5. Add one optional tweak at a time.

> **⚠️ Warning:** If you apply five changes at once (driver, graphics API, mod set, resolution, shader options), you won’t know which change caused instability.
