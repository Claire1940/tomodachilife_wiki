## FAQ

**Q: does tomodachi demo carry over if I used multiple Switch profiles?**

A: It most likely carries over per profile, not globally across the console. If your favorite island is on Profile B, you may need to play the full game on Profile B to use that data.

**Q: Can I keep replaying the demo without deleting my main island?**

A: Yes. Create additional Switch user profiles and start new islands there. This protects your primary save while giving you fresh runs.

**Q: If my demo already ended, is my save still useful?**

A: Yes. Even after progression caps, Mii customization and setup work can still make that save valuable for transfer preparation.

**Q: Should I delete demo data to “fix” transfer issues?**

A: Usually no. Deleting is only smart if you intentionally want a full rebuild. If transfer matters, preserve your main-profile save and avoid unnecessary resets.
