独特なソーシャルシミュレーションシリーズの最新作の発売が待ち望まれる中、多くのプレイヤーが「**トモダチコレクション リビング・ザ・ドリームで何人のMiiを**実際に島に住ませることができるのか？」という疑問を抱いています。これまでの作品とは異なり、今作では島の発展が住民の数と直接結びついた特定の節目（マイルストーン）が導入されています。賑やかな大都市を計画している場合でも、静かな隠れ家を作りたい場合でも、進行には人口の仕組みを理解することが不可欠です。

## 総居住人数と制限

任天堂から公開された公式の日本語FAQによると、1つの島に登録できる住民の最大数は**70人のMiiキャラクター**です。これはオリジナルの3DS版で見られた100人の制限からわずかに減少していますが、開発者はMiiが屋外スペースでより頻繁に交流する「緊密なコミュニティ」の雰囲気を強調しています。

Nintendo SwitchまたはNintendo Switch 2の各ユーザープロファイルは、それぞれ独自のセーブデータを保持します。つまり、1つのプロファイルは70人に制限されていますが、同じ本体の他のユーザーは、独自の70人の名簿を持つユニークな島を新しく開始することができます。

| 特徴 | 制限 / 仕様 |
| :--- | :--- |
| **最大Mii収容数** | 70人 |
| **ユーザーあたりのセーブデータ** | 1つの島 |
| **島の共有** | 不可（ユーザープロファイルに限定） |
| **対応機種** | Nintendo Switch & Nintendo Switch 2 |

> **💡 ヒント:** 島の社会的なダイナミクスを最大化するために、多様な性格を混ぜることを目指しましょう。70人の制限があるため、各住民のユニークな特性がコミュニティ全体のムードに大きな影響を与えるようになります。

## Miiの数による施設の解放

『トモダチコレクション リビング・ザ・ドリーム』では、進行は単なる時間経過ではなく、人口によって決まります。島の最も重要なショップや機能の多くは、特定の住民数に達するまでロックされたままです。序盤のゲームプレイはMiiの頭上に現れる吹き出し「悩み事」に焦点を当てていますが、実際の島の物理的な拡張は、そこに住む人々の数によって引き起こされます。

### 序盤のマイルストーン (1〜5人)

ゲームの最初の数時間は、Miiの世話の基本を学ぶために厳格に制限されています。最初は1人の住民から始まりますが、友人や家族を追加するにつれて島はすぐに開けていきます。

| Miiの住民数 | 解放される施設 / 機能 | 条件 |
| :--- | :--- | :--- |
| **1人目** | 最初の居住区 | 最初のMiiの悩み事を解決する |
| **1人目（レベルアップ）** | 祈りの泉 | いずれかのMiiが初めてレベルアップする |
| **3人目** | 「あちこち」ショップ | 少なくとも5つのMiiの悩み事を解決する |
| **4人目** | TNCルノー | 4人目の住民が引っ越してくる |
| **5人目** | 島デザインセンター | 「クイックビルド」モードとしても知られる |

### 6人のマイルストーン：地形編集

2026年版における最も重要な変更点の一つは、島のカスタマイズ機能の導入です。デモ版と公式プレビューに基づくと、完全なクリエイティブコントロールを得るための魔法の数字は**6人の住民**のようです。6人目のMiiが離れた小さな島エリアに引っ越してくると、島の地形編集と高度なカスタマイズ機能の解放がトリガーされます。

任天堂の内部資料によると、「6人以上の住民がいることで、Miiたちは幸せで楽しい生活を送れるようになる」とされており、ゲーム側が6人を機能的な社会エコシステムの最小単位と考えていることが示唆されています。このポイント以降、ゲームの焦点は人口ベースの解放から「悩み事ベース」の進行へとシフトします。

## Miiのインポートと共有

ファンの間で大きな議論の的となっているのが、Mii共有の制限です。QRコードを使用していた3DS版とは異なり、『リビング・ザ・ドリーム』にはゲーム自体にネイティブな「Mii共有」機能がありません。これは、フィルタリングされていないコンテンツの拡散を防ぐために任天堂が実施した安全策であると報告されています。

しかし、**『Miitopia（ミートピア）』**（製品版または無料のeShop体験版）を使用した確実な回避策があります。お気に入りのキャラクターをゲームに連れてくるには、次の手順に従ってください。

1.  Nintendo Switchで**『Miitopia』**を開きます。
2.  「Miiキャラクター」メニューに移動します。
3.  Miiを選択し、「Nintendo Switch本体へ」を選択します。
4.  『トモダチコレクション リビング・ザ・ドリーム』を起動します。
5.  新しい住民を作成する際、「本体のMiiから作る」を選択します。

> **⚠️ 注意:** MiitopiaからMiiをインポートする場合、特別なメイクやウィッグはコピーされません。複雑な外見を再現するには、新しいゲーム内のフェイスペイントエディタを使用する必要があります。

## 性別、恋愛、そして多様性

2026年版のトモダチコレクションは、その社会システムを大幅に拡張しました。**トモダチコレクション リビング・ザ・ドリームで何人のMii**を作成するかを決める際、ゲームプレイ上の目的で厳格な性別比を気にする必要はもうありません。

*   **性別の選択肢:** プレイヤーは「男性」「女性」「その他」から選択できます。
*   **恋愛の柔軟性:** Miiの恋愛対象を3つの性別の任意の組み合わせに設定したり、恋愛を完全にオプトアウト（除外）したりできます。
*   **家族生活:** 結婚したカップルは、親の性別の組み合わせに関係なく赤ちゃんを持つことができます。

この柔軟性により、過去の「バイナリ（二者択一）」な結婚システムに縛られることなく、70人の制限枠を使って望み通りのコミュニティを表現できるようになりました。

## 島の拡張とスペース管理

島は最初は小さいですが、人口が増えるにつれて拡張されます。チュートリアル中にブイで仕切られている初期スペースは、約6人のMiiと基本的な施設には十分です。全陸地にアクセスするには、住民を追加し続ける必要があります。

| 拡張段階 | およそのMii数 | 主なメリット |
| :--- | :--- | :--- |
| **初期状態の島** | 1-6人 | チュートリアルエリアと基本ショップ |
| **第一次拡張** | 7-15人 | より多くの住宅区画が解放される |
| **中規模の島** | 16-35人 | 高度な地形編集ツールが解放される |
| **最大拡張** | 36-70人 | 最大の建築スペースとランドマークスロット |

## FAQ

**Q: 具体的に『トモダチコレクション リビング・ザ・ドリーム』では何人のMiiを持てますか？**

A: 1つのユーザープロファイルにつき、1つの島に最大70人のMiiキャラクターを登録できます。

**Q: 3DS版の『トモダチコレクション 新生活』からMiiを転送できますか？**

A: いいえ、ニンテンドー3DSから直接転送する方法はありません。作り直すか、Miitopiaのインポート回避策を使用して、まずSwitch本体にベースとなる顔を持ってくる必要があります。

**Q: すべてを解放するために70枠すべてを埋める必要がありますか？**

A: いいえ。ほとんどの主要な施設や地形編集ツールは、住民が6人から10人に達するまでに解放されます。残りのスロットは、個人的な楽しみや、より多様な社会的交流を見るためのものです。

**Q: 島で生まれる赤ちゃんの数に制限はありますか？**

A: 具体的な「赤ちゃんの定員」はありませんが、住民の総数（大人と子供の合計）が70人の登録制限を超えることはできません。島が満員の場合、空きスロットができるまで新しい赤ちゃんは生まれません。
