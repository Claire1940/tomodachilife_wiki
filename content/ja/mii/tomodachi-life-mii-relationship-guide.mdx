## 人間関係のレベルと状態を理解する

トモダチコレクションでは、すべてのMiiが、出会った他のすべてのMiiに対して特定のステータスを持っています。これは、Miiのプロフィールの「人間関係」タブで確認できます。ゲームでは色分けされたフレーズを使って、2人の仲の良さを示します。これらは、恋人同士なら「ちょっと不安」から「運命の人」まで、友達同士なら「うまくいっていない」から「大親友」までの幅があります。

| ステータス タイプ | 関係ラベル | 意味と影響 |
| :--- | :--- | :--- |
| **友情** | 大親友 | 恋愛以外の友情の最高レベル。同居を提案する可能性が高い。 |
| **友情** | 仲良し | 強い絆があり、お互いのアパートを頻繁に訪れる。 |
| **恋愛** | 運命の人 | 恋愛感情の絶頂。プロポーズが成功する可能性が非常に高い。 |
| **恋愛** | 結婚したい！ | Miiがパートナーへのプロポーズを積極的に考えている状態。 |
| **ネガティブ** | うまくいっていない | 喧嘩をしたり、関係が完全に解消されたりするリスクが高い。 |

これらのステータスを改善するには、Miiたちが一緒に過ごす時間を増やすように促す必要があります。Miiに誰かを好きになるよう強制することはできませんが、Wii Uや旅行券などの同じアイテムを与えることで、一緒に過ごすカットシーンを誘発し、会う頻度に影響を与えることができます。

## Miiを同居させる方法

2026年の「Living the Dream」アップデート（およびベースゲーム）で最も求められている機能の一つが、Miiが家を共有する機能です。すべてのMiiが一人で暮らす標準的なアパートとは異なり、共同住宅ではよりコミュニティ的な体験が可能です。しかし、Miiを別のMiiの家にドラッグ＆ドロップするだけで同居させることはできません。このプロセスは完全に有機的であり、関係の進展によって決まります。

### 方法1：結婚
結婚は、Miiを同居させるための最も確実な方法です。2人のMiiが恋愛関係にあり、好感度が十分に高まると、窓に「恋」の吹き出し（ピンクのハート）が表示されることがあります。これは彼らがプロポーズを考えているサインです。プロポーズのミニゲームに成功すると、カップルは結婚し、島にある一軒家に自動的に引っ越します。

### 方法2：ルームメイト（親友）
2人のMiiが「大親友」のステータスに達すると、最終的にルームメイトになることを提案してくることがあります。これは結婚よりも珍しいイベントですが、同様に機能します。プロンプトが表示されたら、彼らが共有スペースに移動することを承認する必要があります。

> **💡 ヒント:** ルームメイトの依頼が発生する確率を高めるには、両方のMiiの幸福度を高く保ち、頻繁に同じ食べ物やギフトを与えて、ゲームのロジック内で「共通の関心事」フラグを立てるようにしてください。

## 世帯の拡大

結婚や友情によって家が設立されると、居住者は2人だけに留まりません。1つの一軒家には最終的に最大8人のMiiが住むことができます。これは、大家族ユニットや、お気に入りのキャラクターたちの「グループホーム」を作るのに非常に便利です。

既存の世帯にさらにMiiを追加するには、新しいMiiがすでにその家に住んでいる少なくとも1人と非常に強い関係を持っている必要があります。プレイを続ける中で、ゲームは主要なライフイベントの際に選択肢を提示することがあります。例えば、ルームメイトが結婚した場合、新しい配偶者を既存のグループホームに迎え入れるか、完全に新しい家を建てるかを選択できます。同様に、子供が生まれたりプロフィール設定から追加されたりした場合、家族全員を一つ屋根の下に留めておくことを選択できます。

| 世帯人数 | 必要条件 | メリット |
| :--- | :--- | :--- |
| **1 Mii** | デフォルト | 標準的なアパート暮らし。管理が容易。 |
| **2 Miis** | 結婚または大親友 | Mii住宅エリアがアンロック。ユニークな交流が発生。 |
| **3-8 Miis** | 家族または強い絆 | 社会的交流が最大化。島のマップのスペースを節約。 |

## 好感度と相性を高める

**トモダチコレクション Mii人間関係ガイド**の核心は「相性診断」です。診断に表示されるパーセンテージ（誕生日に基づく）は、Miiが付き合えるかどうかを厳密に決定するものではありませんが、好感度がどれくらい早く成長するかに影響します。相性が90%であれば、喧嘩を早く解決し、会話中に好感度ポイントをより急速に獲得できる可能性が高くなります。

### Miiの絆を深める主な方法：
1.  **旅行券:** 友達と一緒にいるMiiに旅行券を渡すと、2人で旅行に出かけ、関係が大幅にアップします。
2.  **悩み解決:** 常に「オレンジ色」の吹き出しに対処しましょう。Miiの社会的な悩み（新しい友達を作りたいなど）を助けることは、彼らの交友関係を広げる主要な方法です。
3.  **プレゼントと趣味:** 「使い捨てカメラ」や「凧」などのアイテムをMiiに与えましょう。2人のMiiがこれらのアイテムを一緒に使うと、絆が強まります。
4.  **食べ物の好み:** 食べ物は主に幸福度に影響しますが、お腹がいっぱいで幸せなMiiは、放置されているMiiよりもポジティブな社会的交流を持ちやすくなります。

> **警告:** Miiに「失恋」のアイコン（青いギザギザのハート）が表示されている場合、その関係は終了しています。新しい関係を築けるようになる前に、好物を与えたり元気づけたりして、立ち直るのを助ける必要があります。

## 喧嘩と別れの管理

島のすべての関係が成功する運命にあるわけではありません。Miiの窓に「炎」の吹き出しが表示されることがあり、これは大きな喧嘩を意味します。Miiをなだめて介入しない限り（通常は、お風呂に入れたり好物を与えたりします）、関係は永続的なダメージを受ける可能性があります。

関係が悪化している場合、Miiがパートナーや友人に対して疑問を抱く「曇り」の吹き出しが表示されることがあります。そこで「もっと頑張って」または「諦めて」と伝えることができます。特定の**トモダチコレクション Mii人間関係ガイド**の戦略に従ってカップルを維持したい場合は、常に努力するように促してください。ただし、必ずしも成功するとは限りません。

| 喧嘩レベル | 見た目のサイン | 推奨される行動 |
| :--- | :--- | :--- |
| **些細な口論** | オレンジの吹き出し | 2人のMiiの間の会話を仲裁する。 |
| **大きな喧嘩** | 炎の吹き出し | すぐにMiiに**オルゴール**または**入浴剤**を与える。 |
| **落ち込み** | 青い吹き出し | **超大好物**を与えて、どんよりした気分を晴らす。 |

島の管理に関するさらに高度なヒントについては、レガシータイトルの情報やコミュニティフォーラムがある[任天堂公式サポートページ](https://www.nintendo.co.jp/support/)をご覧ください。

## FAQ

**Q: 2人のMiiを強制的に結婚させることはできますか？**

A: いいえ、結婚を強制することはできません。頻繁に交流するように促し、Miiが自発的にプロポーズを開始した際にミニゲームを成功させることでのみ、関係を後押しできます。

**Q: 相性診断はどうすればアンロックされますか？**

A: 相性診断は、島に少なくとも男性2人と女性2人のMiiが住んでおり、一定回数の社会的交流が行われるとアンロックされます。これはガイド愛好家にとって、定期的にチェックすべき不可欠な要素です。

**Q: 1つの家に住めるMiiの最大人数は何人ですか？**

A: 共有生活を正当化するために必要な家族関係や友情の絆がある場合に限り、1つの家に最大8人のMiiが住むことができます。

**Q: Miiは離婚しますか？**

A: はい。夫婦の関係ステータスが著しく低下し、喧嘩の後に和解できなかった場合、離婚を切り出すことがあります。この場合、家は撤去され、両方のMiiは別々のアパートに戻ります。
