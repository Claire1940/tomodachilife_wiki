<Section icon={Target} title="相性テスターのアンロックと使い方">

**親友の性格相性**を真にマスターするには、相性テスターをアンロックする必要があります。この施設は、島にいる2人のMiiについて、数値によるパーセンテージと天気予報に基づいた予測を提供します。特定の結果を保証するものではありませんが、誕生日と性格特性に基づいて、ペアリングの「生の」可能性を示します。

<Steps>
  <Step num="1" title="アンロック条件を満たす">
    島に男性Miiが2人以上、女性Miiが2人以上住んでいる必要があります。さらに、Miiの悩み（黒またはオレンジの吹き出し）を30個以上解決する必要があります。
  </Step>
  <Step num="2" title="誕生日を確認する">
    テスターは、両方のMiiの完全な誕生日（月/日/年）を必要とします。Miiの誕生日が不明な場合、相性は計算できません。
  </Step>
  <Step num="3" title="ペアリングを選択する">
    テストする2人のMiiを選択します。ゲームは、友情の予測（同性または異なる年齢層）またはロマンスの予測（異性、同年齢層）を提供します。
  </Step>
  <Step num="4" title="結果を解釈する">
    結果はパーセンテージ（0%から100%）と週間予報で表示されます。「晴れ」の予報は絆を深めるのに最適な週を示し、「嵐」の予報は口論の可能性を示唆します。
  </Step>
</Steps>

| 予報アイコン | 意味 | 相性への影響 |
| :--- | :--- | :--- |
| **晴れ** | 素晴らしい | 告白が成功する可能性が高い |
| **曇り** | 平均的 | 着実な関係でわずかな成長 |
| **雨** | 悪い | 誤解が生じやすい |
| **嵐** | 最悪 | 大喧嘩や破局のリスクが高い |

<Callout type="warning" title="テスターの限界">
相性テスターは予測であり、運命ではありません。相性が10%のカップルでも、適切なアドバイスやプレゼントを与えることで結婚できる可能性があります！
</Callout>

</Section>

<Section icon={Swords} title="関係性ステータスラダー">

トモダチライフと2026年版『Living the Dream』では、関係性は特定の「ステータスラベル」を通じて追跡されます。これらのラベルはMiiのプロフィールに表示され、日々の交流に基づいて変化します。これらのレベルを知ることは、Miiがプロポーズを申し出る時期や、友情が終わる危険がある時期を予測するために不可欠です。

| カテゴリ | 高ステータス（緑） | 中ステータス（オレンジ） | 低ステータス（紫） |
| :--- | :--- | :--- | :--- |
| **友情** | 親友 / BFF | 大親友 / 良い友達 | 仲が悪い |
| **ロマンス** | 結婚しよう！ | 超ラブラブ | 不幸 |
| **結婚** | ソウルメイト | 超幸せ | 不幸せ |

**友情ステータスレベル（北米版）：**
1. **親友**: 標準的な友達の最高レベル。
2. **良い友達**: 非常に強く安定した絆。
3. **大親友**: 信頼できる友情。
4. **良い友達**: デフォルトのポジティブなステータス。
5. **まあまあ仲良し**: 中立的な領域。
6. **ちょっと仲良し**: 距離が開き始める。
7. **仲が悪い**: Miiが友達をやめたがるリスクが高い。

<Callout type="success" title="ロマンスの兆候">
恋人関係が「結婚しよう！」または「超ラブラブ」に達した場合、次の数回のゲームプレイセッション中にプロポーズの依頼が表示される可能性が高いです。
</Callout>

</Section>

<Section icon={Shield} title="結婚、破局、離婚">

多くの人にとって、**親友の性格相性**の最終目標は結婚し、家族を築くことです。カップルが結婚すると、Miiホームに引っ越します。しかし、ゲームはシミュレーションであり、関係にはメンテナンスが必要です。結婚ステータスが紫色の「不幸せ」ゾーンに落ちると、カップルは離婚を要求する可能性があり、その場合、彼らはMiiホームから移動し、アパートに戻ります。

**Miiホームへの道：**
1. **告白**: 一方のMiiがもう一方に恋人になってほしいと頼む。
2. **デート**: カップルは一緒に時間を過ごし、ロマンスのステータスを高める。
3. **プロポーズ**: Miiがタイミングミニゲームを通じてプロポーズの手助けを求める。
4. **結婚式**: 式が執り行われ、カップルはハネムーンに行く。
5. **Miiホーム**: カップルは島の永住の家に引っ越す。

<Callout type="warning" title="離婚の警告">
トモダチライフでは離婚は永続的です。Miiは後で再び交際できる可能性はありますが、「元配偶者」としての過去があるため、相性を再構築するのははるかに困難になります。
</Callout>

| イベント | 条件 | 結果 |
| :--- | :--- | :--- |
| **プロポーズ** | 高いロマンスステータス | 結婚＆Miiホームのアンロック |
| **赤ちゃんのお願い** | 結婚済み + ポジティブなステータス | 1〜2日後に子供が生まれる |
| **破局** | 低いロマンスステータス | Miiは「元恋人」になる |
| **離婚** | 低い結婚ステータス | Miiはアパートに戻る |

</Section>
