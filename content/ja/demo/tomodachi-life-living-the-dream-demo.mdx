正式名称「ウェルカム版」として突如サプライズ配信された**『トモダチコレクション Living the Dream』体験版**により、ファンは待望の続編を初めて体験できるようになりました。この体験版では、島生活の初期段階を探索できるほか、刷新されたMiiスタジオ、新しいソーシャルメカニクス、そして本作を象徴する革新的な進行システムを深く掘り下げることができます。3DSのオリジナル版からのファンも、シリーズ初心者の方も、この**『トモダチコレクション Living the Dream』体験版**は製品版への重要な入門ガイドとなるでしょう。本ガイドでは、体験版に含まれるすべての機能、制限、秘密を詳しく解説し、初期の島生活を最大限に楽しむためのヒントをお伝えします。

## はじめに：初期設定と地域の選択

**『トモダチコレクション Living the Dream』体験版**を最初に起動すると、システムの本体日付と時刻のチェックが行われます。ゲーム内のイベントやショップの品揃えは現実の時間と連動しているため、Nintendo Switchの時計を現実の時間に合わせておくことを強くおすすめします。

初期設定における最も重要なステップの一つが、地域の選択です。他の多くの設定とは異なり、地域の選択は確定すると後から変更することができません。この選択によって、島で使用される通貨や、ショップに並ぶ地域限定の食べ物が決定されます。

| 地域のカテゴリー | 含まれる地域 |
| :--- | :--- |
| **北半球** | 日本、ヨーロッパ、北アメリカ、その他 |
| **南半球** | 南アメリカ、オーストラリア、ニュージーランド、その他 |
| **特別地域** | 香港、タイ、韓国 |

地域を選択した後は、島の名前を決めます。島の名前は最大**27文字**まで入力可能で、新機能として「発音」を編集できるようになりました。これにより、ニュース放送や会話の中でMiiたちが島の名前を正しく発音してくれるようになります。

## 刷新されたMiiスタジオとカスタマイズ

**『トモダチコレクション Living the Dream』体験版**のMiiスタジオは、特に芸術的な表現とアイデンティティの面で大幅な刷新が行われました。新しいフェイスペインティング・システムは大きな目玉で、6ページにわたるスタンプが用意されており、カスタムデザインを再利用可能なスタンプとして保存することも可能です。

### 新しいアイデンティティ設定
任天堂は、Miiをよりパーソナルなものにするために、長年要望されていたいくつかの機能を導入しました。
*   **代名詞:** 「彼/彼ら」「彼女/彼女ら」「彼ら（中性的）」から選択できます。
*   **イベント衣装:** 結婚式などのフォーマルなイベントで、男性的な服装、女性的な服装、または中性的な服装のどれを好むかを設定できます。
*   **加齢停止の切り替え:** 誕生日と年齢を個別に設定できるようになり、Miiの加齢を完全に止めるオプションも追加されました。
*   **肌の色の選択:** Miiと触れ合う際に表示される「人間の手」（プレイヤーのカーソル）の肌の色を6種類から選択できます。

| 機能 | 説明 |

| **フェイスペインティング** | 2つのレイヤー（背面/前面）があり、照明の強さをスライダーで調整可能。 |
| **声** | 10種類の基本音声に加え、無限の「ランダム」オプションでユニークなピッチを作成可能。 |

| **プロフィールカード** | Miiのステータスがリアルタイムで更新されるゲーム内の物理アイテム。 |

## 島の発展：「あったかハート」と願いの泉

前作からの大きな変更点は、「あったかハート（Warm Fuzzies）」の導入です。これはMiiの悩み事（今作では「考えごと」と呼ばれます）を解決したり、食べ物を与えたりすることで獲得できる新しい通貨です。お金は依然として**フレッシュ・キングダム**（食品店）や**ウェア・アンド・ウェア**（服屋）でのアイテム購入に使用されますが、「あったかハート」は専ら**願いの泉**で使用されます。

願いの泉は、島の発展の中心的なハブとして機能します。「あったかハート」を寄付することで泉のレベルが上がり、「願い」を獲得できます。製品版ではこれにより主要な施設のほとんどが解放される仕組みですが、体験版では主に新しい**「ちょっとした癖」**や**「お宝」**を解放するために使用されます。

### レベルアップのプレゼント
Miiがレベルアップすると、3つのカテゴリーから報酬を選ぶことができます。
1.  **お宝:** Miiが部屋で使用できるアイテム（例：シャボン玉、マラカス）。
2.  **ちょっとした癖:** 特定の挨拶のスタイルや歩き方など、ユニークな性格特性。
3.  **セリフ:** 特定の感情やイベントの際にMiiが話すフレーズ。

> **💡 ヒント:** 体験版では、島のレベルは最大2までに制限されています。この制限に達するとお金や「あったかハート」を獲得できなくなるため、計画的に使いましょう！

## 社会力学と家系図

**『トモダチコレクション Living the Dream』体験版**では、Mii同士の交流を根本的に変える堅牢な家系図システムが導入されています。新しい住人を作成する際、既存の島民との関係を指定できるようになりました。

以下のオプションが含まれています：
*   兄・姉 / 弟・妹
*   親 / 子
*   祖父母 / 孫
*   親戚

これらの関係を設定することは非常に重要です。なぜなら、設定されたMii同士は**恋愛対象から除外される**からです。これにより、3DS版でよく見られた「気まずい偶然のロマンス」を防ぐことができます。さらに、体験版ではMiiを別のMiiのところまで「ドラッグ」して強制的に紹介させることができ、より早く知り合いにさせることが可能です。

## 体験版の制限とクリア特典

体験版は機能が豊富ですが、いくつかの制限が存在します。これらを把握しておくことで、いつ製品版へデータを引き継ぐべきかの判断に役立ちます。

| 機能 | 体験版の制限 | 製品版の要件 |
| :--- | :--- | :--- |
| **住人の数** | 最大3人 | 100人以上 |
| **島のレベル** | レベル2まで | 無制限 |
| **施設・ショップ** | 食品店 & 服屋 | 10以上の施設 |
| **セーブデータ** | 製品版へ引き継ぎ可能 | N/A |

体験版の主な目標を達成すると、**「ハムスターの服」**がプレゼントされます。この服の色はランダムで決まります。体験版をクリアした後、Miiたちは家の中に留まり、製品版の宣伝フレーズを繰り返すようになります。ただし、毎日ログインしてショップの「日替わり商品」をチェックすることは可能です（ただし、体験版期間中に更新されるのは一度だけです）。

## 新しいミニゲームとランダムイベント

体験版では、Miiのアイコンに紫色の線が表示される「身体的なトラブル」のシナリオがいくつか紹介されています。これはMiiがどこかに挟まったり転んだりしたときに発生します。「人間の手」を使って彼らをつまみ上げ、問題を解決してあげましょう。

新しいミニゲーム**「コインスピン」**も登場しました。このゲームでは、Miiが顔の描かれた巨大なコインを回転させ、プレイヤーが表か裏かを当てます。勝つと3つのギフトバスケット（小・中・大）から一つ選ぶことができ、負けると残念賞としてポケットティッシュがもらえます。

> **⚠️ 注意:** 住人を完全にコントロールしたい場合は、Mii作成時の「おまかせ」機能の使用に注意してください。多くの特徴がランダムに設定されてしまうため、「最初から作る」オプションを使用する方が確実です。

## よくある質問（FAQ）

**Q: 『トモダチコレクション Living the Dream』体験版の進行状況は製品版に引き継げますか？**

A: はい。島の名前、Miiの住人、獲得したアイテム（ハムスターの服など）を含むすべての進行状況を製品版に引き継ぐことができます。

**Q: 体験版では最大何人の住人を登録できますか？**

A: 体験版は最大3人までに制限されています。製品版では、基本的なショップや施設をすべて解放するために少なくとも6人の住人が必要になります。

**Q: 一度決めた島の名前を変更することはできますか？**

A: はい。地域の選択とは異なり、島の名前は「島情報」メニューからいつでも変更できます。Miiの読み方がおかしい場合は、発音を調整することも可能です。

**Q: 同じSwitch内の異なるプロフィールでセーブデータは共有されますか？**

A: いいえ、このゲームのセーブデータは個々のユーザープロフィールに紐付けられています。つまり、Switch内の各アカウントがそれぞれ独自の島を持つことができ、他のタイトルで見られるような「本体に一つの島」という制限はありません。
