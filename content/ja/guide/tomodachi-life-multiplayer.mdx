<Section icon={Map} title="トモダチライフ マルチプレイの進化">

Nintendo Switchにおける『*Tomodachi Life: Living the Dream*』の登場は、プレイヤーが友達の島と交流する方法を根本から変えました。すれちがい通信やいつの間に通信に大きく依存していたオリジナル3DS版とは異なり、2026年版では現代のゲーム機に合わせた堅牢な**トモダチライフ マルチプレイ**オプションが導入されています。基本となる体験は魅力的な放置系生活シミュレーターのままですが、友達の島に直接影響を与えることができるようになったことで、新たな戦略性とユーモアが加わりました。

このバージョンでは、実在の友達、家族、または架空のキャラクターに基づいたMiiたちが暮らす島の「管理人」として行動します。マルチプレイ要素により、2人のプレイヤーがローカルで本体を同期させ、それぞれのMiiが同じ島に住んでいるかのようにどのように交流するかを見ることができます。

| 機能 | 3DS版（オリジナル） | Switch版（2026） |
| :--- | :--- | :--- |
| **接続タイプ** | すれちがい通信 / いつの間に通信 | ローカル通信 / NSO |
| **プレイヤー人数** | 1人（パッシブ） | 2人（ローカル同期） |
| **Mii収容数** | 100人 | 70人（高詳細モデル） |
| **アイテム取引** | 輸入品のみ | フルカスタムアイテム取引 |
| **オンライン共有** | 画像投稿ツール | 統合型ソーシャルフィード |

<Callout type="tip" title="管理人のヒント">

<Section icon={Swords} title="マルチプレイセッションのセットアップ方法">

**トモダチライフ マルチプレイ**セッションのセットアップは簡単ですが、両方のプレイヤーが最初の島のチュートリアルをクリアしている必要があります。マルチプレイメニューにアクセスするには、「市役所」と「港」を解放する必要があります。

<Steps>
  <Step num="1" title="港を解放する">
    島で最初の3つのMiiの悩みを解決し、南海岸にある「港」の施設を解放します。
  </Step>
  <Step num="2" title="無線通信を有効にする">
    両方のプレイヤーが、Nintendo Switch本体の機内モードをオフにし、無線通信が有効になっていることを確認します。
  </Step>
  <Step num="3" title="同期モードを選択する">
    市役所に行き、「ローカル通信」オプションを選択します。一方が「ホスト」を選択し、もう一方が「参加」を選択します。
  </Step>
  <Step num="4" title="データを交換する">
    接続されたら、友達の島に送りたいMiiやアイテムを選択できます。
  </Step>
</Steps>

<Callout type="success" title="高速同期">
港の「クイック同期」機能を使用すると、日替わりアイテム（食べ物や服など）を30秒未満で取引できます。
</Callout>

</Section>

<Section icon={Sparkles} title="マルチプレイのアクティビティとソーシャル共有">

**トモダチライフ マルチプレイ**で接続すると、ソロプレイでは体験できないユニークなインタラクションがいくつか楽しめます。これらはシリーズ特有のシュールなユーモアに焦点を当てており、自分のMiiが親友のMiiと喧嘩をするのを見たり、音楽堂でデュエットを歌う姿を鑑賞したりできます。

**主なマルチプレイアクティビティ：**
- **Miiの旅立ち（訪問）**：Miiを24時間友達の島に滞在させます。
- **プレゼント交換**：珍しいインテリアセットや特別な食べ物アイテムを取引します。
- **カスタムフェイスペイント**：ユニークなフェイスペイントのデザインを他のプレイヤーと共有します。
- **島ツアー**：友達の島のレイアウトやカスタマイズされたショップの品揃えを見学します。

| アクティビティ | メリット | 制限事項 |
| :--- | :--- | :--- |
| **Miiの訪問** | 親密レベルが上昇する | 滞在中は元の島でMiiが不在になる |
| **アイテム取引** | コレクションを完成させやすくなる | 一部のレアアイテムは取引不可 |
| **写真共有** | ソーシャルランクが上昇する | 1日5回までのアップロード制限あり |
| **コンサートデュエット** | 新しい曲が解放される | 両方のMiiがそのジャンルを知っている必要がある |

<Callout type="info" title="カスタマイズの自由度">
2026年版では、カスタムの絵やペットに対する表現規制はありません。ただし、オンラインでMiiを共有する際は、アカウント制限を避けるために任天堂のコミュニティガイドラインに従うことが推奨されます。
</Callout>
