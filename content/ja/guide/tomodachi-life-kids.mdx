<Section icon={Map} title="赤ちゃんの誕生とリクエストの発生">

**トモダチコレクションの子ども**の物語は、赤ちゃんがやってくるずっと前から始まっています。島で新しい命の誕生を目にするには、まず島の設定が許可されているかを確認する必要があります。デフォルトでは赤ちゃんの設定は有効になっていますが、既婚カップルに子どもの気配がない場合は、設定を再確認することをお勧めします。

設定を管理するには、**市役所**へ行き「オプション」から赤ちゃんの設定を確認してください。『Living the Dream』版では、全般オプションメニュー内にある場合もあります。この設定がオフになっていると、新しいリクエストは発生しません（すでに島にいる子どもには影響しません）。有効にすると、リクエストが発生するかどうかは既婚Miiの親密度に大きく左右されます。

| 関係の状態 | 赤ちゃんリクエストの発生率 | 推奨されるアクション |
| :--- | :--- | :--- |
| **かけがえのない人** | 極めて高い | 頻繁にピンクの吹き出しをチェック |
| **満足度MAX** | 高い | 旅行券を渡して絆を深める |
| **とても幸せ** | 中程度 | 毎日コミュニケーションを取る |
| **幸せ** | 低い | プレゼントで関係を改善する |
| **うまくいっている** | 極めて低い | 関係の修復に専念する |

Miiが赤ちゃんを欲しがっているとき、アパートや家の窓にピンクのハートアイコンが表示されます。訪問すると、「赤ちゃんが欲しいと思っている」と相談されます。島のそっくりさんであるあなたは、背中を押すことも思いとどまらせることもできます。励ました場合、約24時間後（『Living the Dream』では最大2.5日後）に誕生の報告が届きます。

<Callout type="tip" title="親密度アップのコツ">
**トモダチコレクションの子ども**の誕生を早めたい場合は、既婚カップルに「旅行券」を使いましょう。一緒に旅行に行くことで親密度が大幅に上がり、その後の数回のプレイセッション内で赤ちゃんのリクエストが発生しやすくなります。
</Callout>

</Section>

<Section icon={Target} title="出産とカスタマイズの仕組み">

予定日が来ると、島を見下ろしている最中に親の片方から電話がかかってきます。これにより、両親が新生児を紹介する特別なカットシーンが始まります。これはプレイヤーにとって非常に重要な瞬間です。なぜなら、子どもが大人になる前にその基本特性に影響を与えることができる唯一の機会だからです。

出産イベント中、子どものアイデンティティを決定する3つの主要な質問をされます：

1.  **性別の希望**: 男の子か女の子かを選ぶか、ゲームに任せることができます。
2.  **見た目**: 両親の特徴を掛け合わせた顔が自動生成されます。そのまま受け入れることも、Miiスタジオに入って細かく調整することも可能です。
3.  **性格**: 両親のどちらかの性格を引き継ぐか、ランダムに生成するか、あるいはスライダーを自分で操作して手動で設定するかを選べます。

<Callout type="warning" title="取り消せない選択">
『Living the Dream』版では、誕生直後に赤ちゃんの顔を編集することはできません。修正を行うには、最初のカットシーンが終わるまで待つ必要があります。初期設定は慎重に行いましょう！
</Callout>

赤ちゃんの性格と性別によって、最初の服の色も決まります。この配色は子供時代のほとんどのステージで維持され、その子の気質を視覚的に表すものとなります。

| 性格タイプ | 男児の服の色 | 女児の服の色 |

| **ナガメ（Outgoing）** | 黄色 | 黄色 |
| **ナゴミ（Easygoing）** | 緑 | 赤 |
| **クール（Independent）** | 青 | 紫 |
| **ドライ（Confident）** | オレンジ | ピンク |

<Section icon={Sparkles} title="5つの成長段階">

**トモダチコレクションの子ども**の成長は非常に速く、現実の時間で約6日間です。毎日、身体的および社会的な発達の大きな節目を迎えます。1日でもログインし忘れると、ステージを飛ばしてしまうことがあるため、すべての瞬間を見守りたい場合は毎日のチェックインが不可欠です。

<Table>
| 日数 | 成長段階 | 身体的変化 | 主な行動 |
| :--- | :--- | :--- | :--- |
| **1日目** | **新生児** | ベビーベッドの中で毛布に包まれている | 泣く、寝る、抱っこされる |
| **2日目** | **はいはい** | 手足を使って床を移動する | 床を探索する、喃語を話す |
| **3日目** | **つかまり立ち** | 立とうとする、髪が生え始める | 転ぶ、しっかり座る |
| **4日目** | **よちよち歩き** | 不安定ながら歩く、幼児の体型 | おもちゃで遊ぶ、泣く回数が減る |
| **5日目** | **子ども** | 髪が揃い、頭の形が定まる | はっきりと話す、走り回る |
| **6日目** | **大人** | Miiのフルサイズに達する | 独り立ちや旅に出る準備ができる |
</Table>

これらの段階の間、両親の家は活動の拠点となります。家を訪ねて、家族のふれあいを見ることができます。興味深いことに、カップルが子育てをしている間は、離婚や大きな喧嘩が発生しなくなり、子どもの成長に安定した環境が保たれます。

<Callout type="success" title="成長の秘密">
ステージ5の子どもが赤ちゃんのような声を出さなくなり、両親の声を高くしたような声で話し始めたら、最後の成長まであと数時間です。アパートの空き部屋を準備しておきましょう！
</Callout>

</Section>

<Section icon={Swords} title="あやし方のミニゲームと交流">

**トモダチコレクションの子ども**を育てるのは体験型のプロセスです。赤ちゃんが泣き止まないとき、両親から助けを求める電話が頻繁にかかってきます。赤ちゃんをうまくあやすことができると、食べ物や宝物などのプレゼントがもらえます。ミニゲームの仕組みは子どもの成長に合わせて変化します。

<Steps>
  <Step num="1" title="ステージ1：揺らす">
    新生児の場合は、3DS/2DSのジャイロ機能を使います。本体を水平に持ち、優しく前後に傾けます。赤ちゃんの頭の周りにキラキラや花が表示されれば成功です。急激な動きをすると両親に怒られるので注意しましょう！
  </Step>
  <Step num="2" title="ステージ2-3：なだめる">
    赤ちゃんが周囲を認識し始めたら、お腹をくすぐったり、頭をなでたり、「いないいないばあ」をしたりします。赤ちゃんごとに好みがランダムに決まっており、くすぐられるのは好きでも「いないいないばあ」は嫌いという子もいます。黒い渦巻きアイコンが出たら、それは嫌がっているサインです。
  </Step>
  <Step num="3" title="ステージ4-5：遊ぶ">
    幼児になると「あやす」ボタンが「遊ぶ」に変わります。タッチスクリーンを使って子どもをぐるぐる回したりできます。この段階では助けを求めて泣くことはありませんが、一緒に遊ぶことで家族全体の幸福度が上がります。
  </Step>
</Steps>

<Callout type="info" title="夜泣きについて">
午後9時以降にゲームを起動し、タイトル画面で泣き声が聞こえたら、赤ちゃんが起きているサインです。家を訪ねると、パジャマ姿で疲れ果てた両親の姿が見られます。手動であやして寝かしつけることに成功すると、家族全員が休めるようになります。
</Callout>

| アクション | 期待される反応 | 最適なステージ |
| :--- | :--- | :--- |
| **くすぐる** | 笑う / おならをする | ステージ2-3 |
| **いないいないばあ** | 驚く / キャッキャと笑う | ステージ3 |
| **なでる** | 落ち着く / 眠くなる | ステージ1-2 |
| **回す** | 目が回る / 喜ぶ | ステージ4-5 |

</Section>

<Section icon={Trophy} title="最後の決断：残るか旅立つか">

6日目、両親から最後のお呼び出しがあります。子どもが大人になり、世に出る準備が整いました。あなたの島の人口構成に関わる重要な瞬間であり、ここでの選択はやり直しができません。

<CardGrid cols={2}>
  <Card title="マンションに住ませる">
    - 子どもは島の**永住者**になります。
    - Miiアパートに空き部屋が必要です。
    - プロフィールや決めゼリフを自由にカスタマイズできます。
    - 他のMiiと交流し、いずれ結婚することも可能です。
  </Card>
  <Card title="旅に出す">
    - 子どもは**旅人**になります。
    - 港から他のプレイヤーの島へと旅立ちます。
    - 旅先から手紙や写真が届きます。
    - 時々、1日だけ両親に会いに帰ってくることがあります。
  </Card>
</CardGrid>

アパートが満室の場合、既存のMiiを削除するか、子どもを旅に出すかの選択を迫られます。同様に、港が旅人で溢れている場合は、スペースが空くまで子どもは出発できません。

<Callout type="warning" title="人口制限">
**トモダチコレクションの子ども**の誕生を待っている間は、常にアパートに1〜2部屋の空きを作っておきましょう。子どもが成長した時に部屋がないと、住人として残す機会を逃してしまいます。これは長期間プレイしている島では特にもったいないことです。
</Callout>

</Section>

<Section icon={Shield} title="Miiの親のためのプロのコツとトリビア">

**トモダチコレクションの子ども**の仕組みをマスターするには、ゲーム内で明示されない隠れた詳細を知る必要があります。遺伝から地域差まで、発見すべきことはたくさんあります。

- **遺伝の仕組み**: 髪の色や顔のパーツは通常、両親の50/50で決まりますが、髪型は完全にランダムか「特徴グループ」に従うことがあります。例えば、母親が髪をまとめている場合、娘もポニーテールや三つ編みになりやすい傾向があります。
- **金色の眉毛の謎**: 元の髪の色に関係なく、ほとんどの赤ちゃんはステージ5まで金色の眉毛をしています。これは、乳児の眉毛が薄くて見えにくいという現実の生物学的特徴を反映したものです。
- **地域による違い**: 北米版や欧州版では「家族の入浴」シーンが削除されています。代わりに、西洋のレーティング基準に合わせて個別のシャワーシーンに変更されています。
- **引き継がれる癖**: 『Living the Dream』Modでは、子どもが親の「ちょっとした癖」や特定の決めゼリフを引き継ぐことがあり、性格の遺伝にさらなる深みが加わっています。
