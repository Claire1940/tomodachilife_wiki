<Section icon={Keyboard} title="2026年における最適なキーボードマッピング">

**トモダチコレクション新生活のキーボード操作**の標準マッピングは、アクセシビリティに重点を置いています。ほとんどのプレイヤーは、左手を移動キーに、右手をマウスに置くことを好みます。この「ハイブリッド」セットアップは、現代のPCゲームの操作方法を反映しています。

  <Card title="移動 (WASD)">
    - **W**: 上へ移動 / 前進

    - **D**: 右へ移動
  </Card>
  <Card title="アクションキー">

  <Card title="セカンダリスティック">

    - **J**: 左
    - **K**: 下

  </Card>

エミュレーターの設定（Ryujinxの入力ウィンドウなど）でこれらを構成する際は、「コントローラータイプ」を **Pro Controller** に設定してください。これにより、エミュレーターが元の3DSボタン数の制限なしに、マップされたすべてのキーを認識します。

| キーカテゴリ | キーバインド | エミュレーター設定 |
| :--- | :--- | :--- |
| **フェイスボタン** | V, B, N, M | A, B, X, Y |
| **Dパッド** | 矢印キー | 上, 下, 左, 右 |
| **システム** | F5 | 更新/リセット |
| **特殊** | Ctrl + F9 | ゲーム終了 |

<Callout type="warning" title="重要">
「ホーム」ボタンを「H」のような一般的なキーにマッピングすることは避けてください。誤って押すと、Citraの一部のバージョンでエミュレーション状態がフリーズする可能性があります。
</Callout>

<Section icon={Target} title="ステップバイステップセットアップガイド">

RMBスクリプト方式を使用して**トモダチコレクション新生活のキーボード操作**を設定することは、マウスとキーボードが調和して機能することを保証する最も信頼性の高い方法です。「Living the Dream」体験のために環境を構成するには、以下の手順に従ってください。

<Steps>
  <Step num="1" title="スクリプトのダウンロードと解凍">
    「RMB Windows」または「RMB Writings」フォルダを見つけます。圧縮ファイルを右クリックし、「**すべて展開**」を選択して設定ファイルにアクセスします。
  </Step>
  <Step num="2" title="コントロールユーティリティの起動">
    解凍したフォルダを開き、実行可能ファイル（通常は `RMB_Writings.exe` という名前）を実行します。このユーティリティは、マウスの動きとエミュレーターのタッチ入力との間のギャップを埋めます。
  </Step>
  <Step num="3" title="ゲームの初期化">
    エミュレーターを起動し、トモダチコレクション新生活をロードします。タイトル画面が表示されたら、RMBユーティリティ内で「**Start**」をクリックして入力を同期します。
  </Step>
  <Step num="4" title="ゲーム内設定の構成">
    エミュレーターの入力設定に移動します。「マウスを有効にする」にチェックが入っていることを確認し、高速な動き中のラグを防ぐために「スティック」感度を100%に設定します。
  </Step>
  <Step num="5" title="テストとキャリブレーション">
    Miiマンションに入り、マウスでMiiの頭をドラッグしてみてください。動きがぎこちない場合は、ユーティリティ設定で「**マウスを隠す**」オプションを切り替えます。
  </Step>
</Steps>

<Callout type="success" title="成功">
設定が完了したら、いつでも **Control + F9** を使用して、ゲームとスクリプトユーティリティを同時に安全に閉じることができます。
</Callout>

</Section>

<Section icon={MousePointer} title="マウスとタッチ操作をマスターする">

トモダチコレクション新生活はスタイラス向けに設計されているため、マウスは実際には**トモダチコレクション新生活のキーボード操作**の最も重要な部分です。多くのプレイヤーは、Miiの頭をなでたり、「くすぐり」ゲームをプレイしたりするのに必要な「スワイプ」動作に苦労しています。

| マウスアクション | ゲーム結果 | 使用シナリオ |
| :--- | :--- | :--- |
| **左クリック** | スタイラスタップ | アイテムの選択、Miiとの会話 |
| **左クリック + ドラッグ** | スタイラススワイプ | Miiをなでる、アイテムを投げる、メニューをスライドさせる |
| **右クリック** | コンテキストメニュー | (一部のスクリプトで) インベントリを素早く開く |
| **スクロールホイール** | ズームイン/アウト | 島や部屋の内部を見る |

落ちてくる物体に瞬時に反応する必要がある「キャッチ」ミニゲームには、高DPIマウスの使用をお勧めします。カーソルが「重い」と感じる場合は、エミュレーターが入力遅延を引き起こす可能性のある**V-Sync**を使用しているかどうかを確認してください。

<Callout type="info" title="技術的な注意">
RMB Writingsスクリプトを使用すると、マウスカーソルの表示を切り替えることができます。カーソルを非表示にすると、ゲーム内の手のアイコンが目に見えないマウスの動きに追従するため、没入感が向上することがよくあります。
</Callout>

</Section>
