- **トモダチコレクションの別れ**イベントは、関係ステータスの低下によって引き起こされ、多くの場合、紫または濃い青の気分色で示されます。
- Miiに「仲直りする」ように指示すれば**和解**は可能ですが、成功は現在の絆レベルに依存します。
- **Miiの悲しみ**は別れの後に続き、悲しみバーを満たすためにプレイヤーが贈り物、食べ物、または注意を与える必要があります。

島のソーシャルシミュレーションという風変わりな世界では、**トモダチコレクションの別れ**は、プレイヤーが目撃する最も劇的なイベントかもしれません。恋人同士の突然の不和であろうと、長年の結婚生活におけるゆっくりとした衰退であろうと、感情的な後遺症を管理することは、ゲームプレイの核となる部分です。このシリーズの人間関係は、目に見えるステータスのはしごによって管理されており、食事を共有することから失敗したデートまで、すべての交流が絆を強めるか、カップルを別れへと押しやります。

2026年にNintendo Switchでリリースされる『*トモダチコレクション：リビング・ザ・ドリーム*』では、これらのソーシャルシステムはさらに微妙になっています。一時的な口論と最終的な関係の破綻の違いを理解することは、あらゆる島管理者にとって不可欠です。このガイドでは、失恋のメカニズム、和解の手順、そして島の生活の激動の社会的水域を通して、お気に入りのMiiカップルを一緒に保つ方法を解説します。

<Section icon={Heart} title="人間関係のはしごを理解する">

ゲーム内のすべてのロマンチックなペアリングは、特定のステータスレベルにあります。北米版および2026年の国際版では、これらのレベルは色分けされており、関係の健全性を視覚的にすばやく把握できます。Miiのステータスが下位層に落ちると、**トモダチコレクションの別れ**のリスクは指数関数的に増加します。

| ステータスレベル (NA) | 色の指標 | 別れのリスク | 意味と影響 |
| :--- | :--- | :--- | :--- |
| **結婚しよう！** | 明るい緑 | 極めて低い | 恋人同士の最高の絆。プロポーズする可能性が高い。 |
| **超ラブラブ** | 緑 | 非常に低い | 安定した幸せな関係で、頻繁にデートする。 |
| **完全にラブラブ** | 薄緑 | 低い | 確固たるつながり。Miiはお互いの会社を楽しむ。 |
| **とてもラブラブ** | 黄緑 | 中程度 | ほとんどのアクティブなカップルにとって標準的な「健全な」レベル。 |
| **ラブラブ** | 黄色 | 中程度 | 中立的な状態。スライドを防ぐために注意が必要。 |
| **仲良し** | オレンジ | 高い | 最初の警告サイン。Miiが口論を始める可能性がある。 |
| **ちょっと仲良し** | 青 | 非常に高い | 危機的な段階。別れのバブルがすぐに出現する可能性がある。 |
| **仲が悪い** | 濃い紫 | 即時 | 最終段階。介入なしでは別れがほぼ確実。 |

<Callout type="warning" title="色に注意">
Miiのプロフィールにある「人間関係」タブを常に確認してください。青や紫が見える場合、カップルは「危険ゾーン」にいます。**トモダチコレクションの別れ**は、片方のMiiがそのように感じているだけでも起こる可能性があります。ゲームは関係の両側を独立して追跡するためです。
</Callout>

人間関係は16種類の性格タイプにも影響されます。たとえば、「気楽な」おっとり型は、そのスライダーが正反対の場合、「自信家」の行動派とより苦労するかもしれません。性格グループを一致させる（例：2人の社交的なMii）と、一般的にプレイヤーの手動での努力が少なくても、より高い絆レベルを維持できます。

</Section>

<Section icon={Swords} title="別れイベント：リクエストの処理">

別れは、Miiが恋愛関連の悩みバブル（ギザギザのハートが付いたピンクのアイコン）を表示したときに始まります。アパートに入ると、Miiはうまくいっていないことを告白し、関係を終わらせるべきかどうか尋ねます。これがあなたの介入の主なポイントです。

**ビデオのハイライト：**
- 「恋愛問題」バブルの特定。
- 別れのリクエスト中の対話選択肢のナビゲート。
- 和解の成功と失敗の視覚的なシーケンス。

<Steps>
  <Step num="1" title="最初の対決">
    Miiが「[名前]と仲良くできていない。どうすればいい？」と尋ねるとき、選択肢が与えられます。関係を救おうとするには、**「仲直りする」**または**「仲直りしよう」**を選択します。
  </Step>
  <Step num="2" title="和解の試み">
    Miiはその後、中立的な場所（公園やカフェなど）でパートナーと会います。彼らは問題を話し合おうとします。『リビング・ザ・ドリーム』（2026年）では、これらのシーンはMiiの特定の性格に基づいた更新された対話が特徴です。
  </Step>
  <Step num="3" title="結果の確認">
    和解が成功した場合、関係ステータスはより高い、より肯定的なレベル（通常は「とてもラブラブ」）にリセットされます。失敗した場合、Miiは正式に別れ、両方とも悲しみバーを持ってアパートに戻ります。
  </Step>
</Steps>

<Callout type="tip" title="リセットルール">
和解が成功した場合、それは関係の負の勢いを「ハードリセット」する役割を果たします。これは、紫色のステータスレベルに陥ったカップルを救う最も効果的な方法です。
</Callout>

</Section>

<Section icon={Shield} title="離婚と結婚の複雑さ">

恋人同士の**トモダチコレクションの別れ**はよくあることですが、結婚したMii間の離婚ははるかに重要です。結婚したカップルは「Miiハウス」に住んでおり、離婚は即座にその家を破壊することになります。恋人とは異なり、結婚したMiiには「配偶者」ステータスのはしごがあり、これは少し回復力がありますが、一度壊れると回復がより困難になります。

| 特徴 | 恋人同士の別れ | 結婚したMiiの離婚 |
| :--- | :--- | :--- |
| **トリガー** | 恋愛問題バブル | 恋愛問題バブル |
| **居住状況** | アパートに滞在 | 家は削除される |
| **悲しみレベル** | 中程度から高い | 極めて高い |
| **子供** | 該当なし | 子供はアパート/旅人になる |
| **元恋人/元配偶者のステータス** | 再び友達になれる場合がある | ほとんどの場合、永久に「元配偶者」のまま |

<Callout type="warning" title="子供の制限">
*トモダチコレクション*では、結婚したMiiは、赤ちゃんを積極的に育てている間は離婚を開始できません。ゲームは、子供が完全に成長し（6日目）、家を出るか旅人になるまで、離婚オプションをロックします。
</Callout>

離婚が発生した場合、カップルの家に贈られたアイテムやインテリアはすべて失われます。両方のMiiはMiiアパートに戻り、かなりのプレイヤーの相互作用を必要とする最大の悲しみバーを持つことになります。2026年Switch版では、「元配偶者」ステータスが、ユニークな「気まずい遭遇」ニュースレポートにつながることがあります。

</Section>

<Section icon={Sparkles} title="別れを防ぐ方法">

予防は、傷ついた心を癒すよりも常に簡単です。「相性診断」を監視し、特定のアイテムを使用することで、関係ステータスを緑色に保つことができます。

<CardGrid cols={3}>
  <Card title="旅行券">
    最も強力なツールです。関係にあるMiiに旅行券を渡すと、パートナーと一緒に休暇に行かせることができ、絆を1～2レベル瞬時に高めます。
  </Card>
  <Card title="恋愛アドバイス">
    Miiが「私と[名前]についてどう思いますか？」と尋ねたときは、常に**「最高の相性！」**を選びましょう。これにより、隠されたソーシャルスコアに小さくても一貫したブーストが与えられます。
  </Card>
  <Card title="相性チェック">
    相性診断（解決した問題が30になるとアンロック）を使用して、毎週の予測を確認しましょう。「恋愛」の予測が嵐の場合、その週は大きな社会的な変化を避けましょう。
  </Card>
</CardGrid>

<Callout type="success" title="性格要素">
性格診断で「動き」と「話し方」のスライダーが似ているMiiは、口論が少ない傾向があります。カップルを一緒にさせたいと切望している場合は、タウンホールで片方のMiiの性格特性をパートナーとより一致するようにわずかに調整することを検討してください。
</Callout>

</Section>
