## 基礎：友達から恋人へ

ウェディングベルについて考える前に、Miiたちはロマンチックな基礎を築かなければなりません。『トモダチコレクション 新生活』における結婚は、段階的な進行システムになっています。「交際」フェーズを無事に通過していない2人のMiiを、無理やり結婚させることはできません。

まず、2人のMiiが友達になる必要があります。強固な友情が築かれると、やがて一方のMiiがもう一方に対して感情を抱くようになります。これは、マンションの窓にピンク色の「ハート」の悩み吹き出しが表示されることで分かります。これを見かけたら、Miiはあなたに告白のアドバイスを求めてきます。告白が成功すれば、彼らは正式にカップル（恋人）になります。

### 関係ステータスの段階

| 段階 | 必要な条件 | 結果 |
| :--- | :--- | :--- |
| **友達** | 最初の交流と紹介の成功。 | Miiがお互いの部屋を頻繁に訪れるようになる。 |
| **恋人** | 愛の告白の成功。 | Miiがデートに行き、お互いのステータス画面に表示される。 |
| **婚約中** | プロポーズ・ミニゲームの成功。 | すぐに結婚式が執り行われる。 |
| **結婚** | 結婚式と新婚旅行の完了。 | Miiたちが一緒に「Miiマイホーム」エリアに引っ越す。 |

> **💡 ヒント:** Miiが告白を緊張している場合は、「ロマンチックに」というアプローチを勧めたり、「手鏡」などのアイテムをプレゼントして自信をつけさせてから送り出しましょう。

## 絆を深める

Miiたちが付き合い始めたら、関係レベルを上げる必要があります。これはMiiのプロフィールから人間関係リストを見ることで確認できます。「かけがえのない人」や「結婚したい！」といったフレーズは、プロポーズが間近であることを示しています。関係が単に「満足」や「仲良し」程度であれば、まだやるべきことがあります。

絆を早く深めるには、頻繁な交流を促しましょう。一人のMiiが寂しがっていたり退屈していたりする場合は、もう一人のMiiを部屋に「引き込む」（比喩的に、一緒に遊ぶプロンプトに答えることで）ようにします。特定のアイテムを提供することもプロセスを加速させます。

**「旅行券」**は、あなたの武器の中で最も効果的なツールです。交際中のMiiに旅行券を渡すと、多くの場合、パートナーをロマンチックな休暇に連れて行きます。これらの旅行は、質の高い思い出、ロマンチックな写真、そして関係スコアの大幅な上昇を生み出します。

## プロポーズ：ミニゲームをマスターする

Miiが結婚を申し込む準備ができると、再びピンク色のハートの吹き出しが現れますが、今回は具体的にパートナーと結婚したいと言い出します。これがプロポーズイベントのトリガーです。演出が中心の告白とは異なり、プロポーズにはプレイヤーの積極的な参加が必要です。

### プロポーズ・ミニゲームの仕組み

このミニゲームでは、あなたはMiiの「心の声」や「タイミングのコーチ」の役割を担います。レストランや公園など、カップルが一緒に座っているシーンが表示されます。あなたの目標は、Miiがパートナーのことを考えている瞬間に「今だ！」ボタンをタップ（またはAボタンを押す）することです。

1.  **吹き出しを観察する:** Miiの頭上に、様々なアイテムや他のMiiを表示する吹き出しが浮かびます。
2.  **パートナーの顔を待つ:** 吹き出しにパートナーの顔が表示されるまで待たなければなりません。
3.  **タイミングがすべて:** パートナーの顔が見えた瞬間に、正確にハートのアイコンをタップします。
4.  **ライフを管理する:** 通常、3回のチャンスがあります。3回タイミングを外すと、Miiは勇気を失い、プロポーズは失敗に終わります。

| 要素 | アクション | 結果 |
| :--- | :--- | :--- |
| **パートナーの顔** | すぐに「今だ！」をタップ。 | プロポーズメーターが進む。 |
| **邪魔なもの（食べ物、趣味）** | タップしない。 | ペナルティはないが、時間の無駄になる。 |
| **違うMiiの顔** | タップしない。 | ここでタップすると失敗とみなされる。 |
| **空の吹き出し** | タップしない。 | ここでタップすると失敗とみなされる。 |

> **⚠️ 注意:** 吹き出しが速く動き始めても、慌てないようにしましょう。焦ってライフを失うよりも、1サイクル見送って確実なチャンスを待つ方が賢明です。

## 結婚を成功させるための条件

**トモダチコレクションでの結婚方法**のすべてのステップに従っていても、ゲームのロジックに組み込まれたハードルによってイベントが発生しない場合があります。

*   **年齢の互換性:** 両方のMiiが同じ年齢層である必要があります。『トモダチコレクション 新生活』では、入力された生年月日に基づいて、Miiは「子供」か「大人」に分類されます。大人同士、子供同士（「大人スプレー」を使用して一時的に大人になって式を挙げる）は結婚できますが、大人と子供は結婚できません。
*   **関係レベル:** 関係ステータスは理想的には「満足」以上であるべきです。それ以下のレベルでも可能ですが、「結婚」の悩み吹き出しが出る確率は非常に低くなります。
*   **マンションの空き:** 島の人口が最大になっていないか確認してください。ただし、通常はMiiマイホームへの移動が優先されます。

公式のメカニズムやアップデートの詳細については、レガシータイトルの情報について[任天堂の公式サポートページ](https://www.nintendo.com/)を確認してください。

## 結婚後の生活

プロポーズが受け入れられると、すぐに結婚式が始まります。結婚式の衣装を着たMiiたちのモンタージュが流れ、その後にランダムな海外の場所での新婚旅行シーケンスが続きます。帰島後、いくつかの変化が起こります。

1.  **Miiマイホーム:** 島の新しいエリア「Miiマイホーム」が有効になります。結婚したカップルはここの一軒家に引っ越します。
2.  **二重生活:** Miiたちはメインのマンションにも個別の部屋を持ち続けますが、ほとんどの時間（特に夜間）を共有のマイホームで過ごすようになります。
3.  **赤ちゃん:** 結婚は「赤ちゃん」メカニズムの前提条件です。しばらくすると、夫婦から子供を持っていいか相談されることがあります。

### 結婚後のメリット

| メリット | 説明 |
| :--- | :--- |
| **新しいアイテム** | 「旅行のお土産」（お宝）を受け取ることができ、銀貨や金貨のように高値で売却できます。 |
| **安定性** | 結婚したMiiは「別れ」に繋がる喧嘩をしにくくなりますが、時々喧嘩の解決が必要な場合もあります。 |
| **島の発展** | 夫婦の間に生まれた子供は、最終的に永久の住人になることができ、島の人口を増やせます。 |

## トラブルシューティング

**トモダチコレクションでの結婚方法**で苦労している場合、その原因の多くは吹き出しのRNG（乱数生成）にあります。

**Q: Miiたちが数週間付き合っていますが、プロポーズしてくれません。何が悪いのでしょうか？**
相性診断で彼らの相性をチェックしてみてください。スコアが低くても結婚が不可能なわけではありませんが、「結婚」の悩み吹き出しが出る頻度がかなり低くなります。共通の幸福度を高めるために、二人が好むプレゼントをもっと贈ってみてください。

**Q: 離婚を止めることはできますか？**
はい。紫色の「悲しい」吹き出しが出ている場合、Miiたちは別れや離婚を考えている可能性があります。すぐに介入して喧嘩を解決してください。この段階で旅行券を渡すと、結婚生活を救えることがあります。

## FAQ

**Q: トモダチコレクションでMiiを早く結婚させるにはどうすればいいですか？**

A: プロセスを早めるには、両方のMiiに旅行券を渡して関係ステータスを最大レベルまで上げてください。また、彼らが出すあらゆる「悩み」吹き出しに対応してください。小さな問題を解決することは、全体的な幸福度を高め、プロポーズの吹き出しが出現する可能性を上げます。

**Q: 年齢が違うMii同士で結婚できますか？**

A: いいえ、大人のMiiと子供のMiiは結婚できません。子供のMiiを別の大人住人と結婚させたい場合は、「大人スプレー」というアイテムを使用して大人に変える必要があります。

**Q: プロポーズのミニゲームに失敗するとどうなりますか？**

A: 失敗すると、Miiは落ち込み、関係レベルが少し下がります。彼らを元気づけ、再びプロポーズの吹き出しが出るのを待つ必要があります。これには数日、あるいは数週間かかることもあります。

**Q: 1人のMiiは何回結婚できますか？**

A: Miiは何度でも結婚できますが、その前に離婚しているか、死別している必要があります。前の関係が正式に終わっていれば、島で何度でも愛を見つけることに制限はありません。
