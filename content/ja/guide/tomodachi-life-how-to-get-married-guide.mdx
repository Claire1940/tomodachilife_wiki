あなたのMiiたちが交流し、ユニークな絆を育むのを見るのは、ニンテンドー3DSの名作の醍醐味です。島の人間関係を次のレベルへと進めたいなら、この**トモダチコレクション 新生活 結婚ガイド**を理解することは、永続的な関係を築く上で不可欠です。ゲーム内の結婚は単なる見た目の変化ではありません。Miiホームや子育てといった新しいゲームプレイ機能をアンロックします。

この**トモダチコレクション 新生活 結婚ガイド**では、友情の最初のきらめきから最後の「誓います」まで、全プロセスを順を追って説明します。お気に入りのMiiたちが互いに気づかないことに苦労している場合でも、プレッシャーの高いプロポーズミニゲームで失敗し続けている場合でも、これらの戦略は恋愛を成功させるのに役立ちます。これらのステップに従うことで、2026年もあなたの島が幸せなカップルと成長する家族でにぎわう中心地であり続けることを保証できます。

## ステップ1：ロマンスの土台を築く

Miiが結婚を考える前に、まず将来のパートナーと友達になる必要があります。2人のMiiを強制的に恋に落ちさせることはできませんが、彼らの性格や日々の交流を管理することで、その可能性に大きく影響を与えることができます。

### 性格と相性
すべてのMiiには、動き、話し方、表現力、態度、全体という5つの特定の特性に基づいて、16種類の性格タイプのいずれかが割り当てられます。これらの性格は、**おっとり**、**ノリノリ**、**クール**、**まじめ**という4つの主要なカテゴリーに分類されます。見た目や声は美的要素ですが、これらの特性がMiiたちの相性を左右します。

| カテゴリー | サブタイプ | 主な特徴 |
| :--- | :--- | :--- |
| **おっとり** | ふんわり、おせっかい、のんびり、どっしり | 共感的、思慮深く、前向き。 |
| **ノリノリ** | テキパキ、がんこ、わがまま、ドライ | 明るく、機知に富み、積極的。 |
| **クール** | こだわり、マイペース、一匹狼、個人主義 | 創造的、自立的、内省的。 |
| **まじめ** | がんばりや、ムードメーカー、ロマンチスト、カリスマ | 勤勉、野心的、知的。 |

> **💡 ヒント：** 2人のMiiが出会う可能性を高めるには、アパートを横に並べて配置してみてください。隣接する部屋に住むMiiは、お互いを訪問し、紹介を頼む可能性が高くなります。

### 友情を育む
Miiがオレンジ色の吹き出しを出している場合、別のMiiに会いたがっている可能性があります。常にこれらの紹介を促しましょう。また、両方のMiiに同じアイテムやプレゼント（同じWii U本体や裁縫セットなど）を与えることで友情を育むことができます。これにより、彼らがアパートで一緒に過ごすことを促します。

## ステップ2：告白（恋人になる）

2人のMiiが友達になると、やがてどちらかが恋心を抱くようになります。これは、アパートの窓に**ピンクのハートアイコン**が表示されることで示されます。これは、この**トモダチコレクション 新生活 結婚ガイド**における最初の大きな節目です。

ハートをタップすると、Miiは自分の気持ちを告白し、あなたにアドバイスを求めてきます。告白のスタイルと場所を選ぶことができます。

| 告白スタイル | 説明 |
| :--- | :--- |
| **ストレート** | 率直で誠実な愛の告白。 |
| **ロマンチック** | 詩的なアプローチで、しばしば花のような言葉遣い。 |
| **必死** | プレッシャーの高い、感情的な愛情の訴え。 |
| **かわいく** | 遊び心があり、魅力的な誘い方。 |
| **プレゼントを渡す** | 食べ物や宝物を使って相手の心をつかむ。 |

### 告白の場所
ビーチ、タワー、公園、学校などの場所を選ぶことができます。告白が成功すると、相手のMiiは受け入れ、関係は「恋人」（米国版）または「特別な人」（欧州版）に変わります。もし別のMiiが告白を邪魔して自分の愛を宣言した場合、ターゲットのMiiは2人のうちどちらかを選ぶか、両方を拒否します。

## ステップ3：関係を深める

Miiたちは、付き合い始めたからといってすぐに結婚できるわけではありません。関係レベルが十分な段階まで進む必要があります。これはMiiの関係プロフィールを見ることで確認できます。「最高の愛」や「結婚しよう！」（米国版）といったフレーズを探しましょう。

このプロセスを早めるには：
1.  **旅行券：** Miiに旅行券をあげましょう。ほとんどの場合、恋人を連れて旅行に行き、関係ステータスに大きな後押しを与えます。
2.  **毎日のチェックイン：** カップルが抱える問題（黒または緑の吹き出し）に対処しましょう。喧嘩している場合は、関係が悪化するのを防ぐため、すぐに解決してあげてください。
3.  **相性テスター：** 主に誕生日に基づいています。これを使ってカップルにとって「恋愛」運が最も高い日を確認しましょう。

## ステップ4：プロポーズミニゲームをマスターする

関係が最高潮に達すると、再びピンクのハートが現れます。今度はMiiがプロポーズしたいと宣言します。これが**トモダチコレクション 新生活 結婚ガイド**の最もインタラクティブな部分であるプロポーズミニゲームの引き金となります。

### 会場を選ぶ
4つの場所から1つを選ぶように求められます。それぞれにユニークな「妨害」があり、ミニゲームをより難しくする可能性があります。

| 場所 | お祝いのスタイル | よくある妨害 |
| :--- | :--- | :--- |
| **花火大会** | ハート型の花火 | 大きな爆発音、他の参加者の叫び声。 |
| **駅** | 観衆の拍手 | 通過する電車、構内アナウンス、咳。 |
| **遊園地** | ロマンチックな音楽 | 騒がしい乗り物、叫び声。 |
| **高級レストラン** | 親密な歓声 | ウェイトレスが皿を落とす音、他の客。 |

### ミニゲームの勝ち方
目的は、恋人がプロポーズする人のことを考えているときに、タッチスクリーン上の**「今だ！」**ボタンをタップすることです。
*   恋人の頭の上に、循環する画像（食べ物、他のMii、趣味）の思考バブルが表示されます。
*   バブルの中に**プロポーズする人の顔**が現れるまで待ちます。
*   素早く「今だ！」をタップします。プロポーズを完了するには、4回成功する必要があります。
*   あなたは3つの「ライフ」（ハートで表示）を持っています。間違ったタイミングでタップしたり、妨害が発生したりすると、ハートが壊れます。

> **⚠️ 注意：** プロポーズミニゲームに3回失敗しても、Miiが悲しむことはありませんが、プロポーズは失敗します。再度ハートアイコンが表示されるまで待って、やり直す必要があります。

## ステップ5：結婚式とその先

プロポーズが受け入れられると、すぐに結婚式が行われます。もしMiiが子供の場合、ゲームは式典の前に大人にするための**大人になるクスリ**を提供します。

### 結婚式
結婚式では、タキシード姿の新郎と白いウェディングドレス姿の新婦が登場します。彼らは誓いを交わし、ケーキをカットし、新婦は島の他の住民にブーケを投げます。式典の後、クレジットが流れる間、彼らの関係のハイライトのスライドショーを見ることができます。

### 結婚生活の機能
*   **Miiホーム：** カップルは島の「Miiホーム」セクションの家に引っ越します。彼らは訪問のためにアパートも保持していますが、ほとんどの時間を家で一緒に過ごします。
*   **お土産：** 新婚旅行（結婚式の直後に行われます）の後、彼らはあなたにユニークな宝物を持ってきてくれます。
*   **赤ちゃん：** 役場で「赤ちゃん」の設定をオンにしている場合、結婚したカップルはやがて子供を授かりたいと尋ねてきます。これは通常、結婚後数日または数週間後に、彼らの関係が良好であれば起こります。

## よくある質問

**Q: 「トモダチコレクション 新生活」では、同性のMiiは結婚できますか？**

A: 「トモダチコレクション 新生活」のオリジナルバージョンでは、結婚は男性と女性のMiiの間でのみ可能です。プレイヤーはMiiの「性別」設定を外見はそのままに変えることで、創造的な回避策を見つけていますが、ゲームのエンジンは同性婚をネイティブにはサポートしていません。

**Q: Miiがプロポーズを拒否したらどうなりますか？**

A: プロポーズミニゲームで3つのハートをすべて失うと、恋人は立ち去ることを求めます。しかし、告白の失敗とは異なり、プロポーズしたMiiは通常「悲しみ」ゲージを得ることはありません。ピンクのハートアイコンが再び表示されれば、いつでも再挑戦できます。

**Q: 結婚したMiiは離婚できますか？**

A: はい、できます。結婚したカップルの仲が悪くなり、関係ステータスが水色または「仲が悪い」という色にまで下がると、一緒に過ごすのをやめることがあります。最終的に、どちらかのMiiが別れや離婚を求めることがあります。もし家に赤ちゃんがいる場合、赤ちゃんが完全に成長するまでは離婚できません。

**Q: 子供のMiiを結婚させるために「大人になるクスリ」はどうやって手に入れるのですか？**

A: 探す必要はありません！2人の子供のMiiがプロポーズを成功させると、ゲームが自動的に「大人になるクスリ」を使うよう促します。式典のために彼らが大人であることを保証するため、結婚式のシーケンスの一部として無料で提供されます。
