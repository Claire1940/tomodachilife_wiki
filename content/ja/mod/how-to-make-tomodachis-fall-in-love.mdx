## Miiの恋愛における必須条件

**トモダチを恋に落ちさせる方法**を実践する前に、選んだMiiたちが恋愛可能な状態であるかを確認する必要があります。ゲームには、恋愛が始まる前にブロックされてしまう年齢や性別の好みに関する厳格なルールが存在します。

### 年齢グループと恋愛対象
『トモダチコレクション』では、Miiは主に2つの年齢グループに分類されます。**子供**（18歳未満）と**大人**（18歳以上）です。Miiの年齢は、作成時に入力した生年によって決定されます。

| Mii Aの状態 | Mii Bの状態 | 恋愛は可能？ |
| :--- | :--- | :--- |
| **大人** | **大人** | はい |
| **子供** | **子供** | はい |
| **大人** | **子供** | **いいえ**（完全にブロックされます） |
| **親族** | **親族** | **いいえ**（家族設定されている場合） |

> **注意：** 2人のMiiを結婚させて子供を作らせたい場合は、両方が「大人」カテゴリーである必要があります。子供同士でも「恋人」にはなれますが、「大人スプレー」を使うか、自然に年齢が上がるまで結婚はできません。

### 恋愛設定の変更
2026年の「ドリーム・ライフ」版では、恋愛の好みを拡張設定できます。Miiを作成または編集する際、誰に惹かれるかを設定できます。もしMii Aの好みにMii Bの性別が含まれていない場合、恋愛関係は決して発生しません。これらの設定は、「市役所」の住人リストから、またはアパートでMiiに直接話しかけることでいつでも調整可能です。

## ステップ・バイ・ステップ：トモダチを恋に落ちさせる方法

Miiが恋愛対象であることを確認したら、関係を築くための積極的なプロセスを開始しましょう。恋愛の火種を作る確率を高めるための戦術的なステップは以下の通りです。

### 1. 最初の出会いをセッティングする
「他人」の状態のMiiが恋に落ちることはありません。まずは紹介される必要があります。Miiのアイコンを会わせたい住人のところへ**ドラッグ**することで、強制的に交流を促すことができます。Miiの頭から青や紫の「イベント」パーティクルが出ている場合は、社会的な交流が行われている合図です。

### 2. 戦略的なアパートの配置
Miiはすぐ隣の住人と交流する可能性が高くなります。**トモダチを恋に落ちさせる方法**を最適化するには、ペアにしたい2人のMiiを、隣同士や真上・真下の部屋に移動させましょう。この近接性により、一緒に過ごす頻度が高まり、自然と友好度が上がります。

### 3. 共通の趣味とレベルアップ時のプレゼント
Miiがレベルアップした際、道具をプレゼントできます。特定の2人の絆を深めるには、全く同じ道具を与えてください。これにより、お互いのアパートでその道具を一緒に使う確率が高まります。

| アイテムカテゴリー | 例 | 恋愛への効果 |
| :--- | :--- | :--- |
| **電化製品** | Wii U、ノートパソコン、ゲーム機 | 一緒に遊ぶセッションが発生しやすい。 |
| **アウトドア** | シャボン玉、凧、サッカーボール | 公園や砂浜での出会いを促す。 |
| **リラックス** | ヨガマット、本、オルゴール | 穏やかな交流と深い絆を深める。 |

## 「恋愛の悩み」イベントを発生させる

**トモダチを恋に落ちさせる方法**として最も直接的なのは、アパートの窓に表示される**恋愛の悩み**（ピンクのハートアイコン）または**恋の橋渡し**（オレンジの吹き出し）です。

### 恋の橋渡し戦略
時折、第三者のMiiがオレンジの吹き出しで「[Miiの名前]に誰か似合う人はいないかな？」と相談してくることがあります。
1. 相談された際、ゲームはランダムな候補を提案します。
2. 提案された候補が希望の人でない場合は、**「他の人がいい」**を選択してください。
3. ペアにしたい特定のMiiを選択します。
4. このセッティングの成功率は、既存の友好度（人間関係メニューの緑色の濃さ）に依存します。

### 告白の管理
Miiが誰かを好きになると、告白の仕方についてアドバイスを求めてきます。場所（砂浜、屋上、公園）と雰囲気（ロマンチックに、ストレートに、かわいく）を選択できます。「ロマンチックに」が最も無難な選択肢であることが多いですが、告白の成功率は決して100%ではありません。失敗した場合、Miiは「落ち込み」状態になります。好物を与えたり、新しい服をプレゼントしたりして解決してあげましょう。

> **ヒント：** Miiが「困っている」イベント（どこかに挟まった、助けが必要など）で、意中の相手に助けてもらうと、稀に即座に「片思い」状態がトリガーされることがあります。

## 高度な関係の維持

恋に落ちることは始まりに過ぎません。結婚や出産という最終目標に到達するには、関係の勢いを維持する必要があります。

### 旅行券の活用
旅行券は、関係ステータスを上げるための最も強力なツールです。2人のMiiがすでに「恋人」である場合、片方に旅行券を渡すと、多くの場合パートナーを旅行に誘います。これにより関係の質が大幅に上昇し、「結婚したい！」というプロンプトに近づきます。

### 三角関係への対処
Miiが同時に2人の人を好きになることがあります。この場合、どちらを追いかけるべきか相談されます。これは、あなたが好むペアリングを固めるチャンスです。最初の相手を選べば、二人目の相手との関係はプラトニックなままか、「知り合い」に格下げされます。

| 関係ランク | 視覚的なインジケーター | 条件 |
| :--- | :--- | :--- |
| **知り合い** | 薄い緑 | 最初の出会い。 |
| **親友** | 濃い緑 / キラキラ | 頻繁な交流 ＋ 共通のアイテム。 |
| **恋人** | ピンクのハート | 告白の成功。 |
| **結婚** | 結婚指輪 | 高い「恋人」ステータス ＋ プロポーズ成功。 |

## よくある質問（FAQ）

**Q: トモダチをより早く恋に落ちさせる方法は？**

A: 最も早い方法は、二人の交流時間を最大化することです。アパートを隣同士にし、同じレベルアップアイテム（ノートパソコンやWii Uなど）を与え、「親友」ステータスに達したら旅行券を使って恋愛への移行を促しましょう。

**Q: 性格が違っても恋に落ちますか？**

A: はい。似た性格（例：二人とも「ナゴミ系」）の方が仲良くなりやすい傾向はありますが、『トモダチコレクション』では「正反対のタイプが惹かれ合う」シナリオも可能です。性格は、恋愛の相性よりもプレゼントや遊びへの反応に影響します。

**Q: Miiがすでに結婚している人を好きになった場合はどうすればいいですか？**

A: これは「三角関係」または「報われない愛」のシナリオになります。Miiが告白を試みることもありますが、相手の結婚生活が「最悪」な状態（人間関係バーが濃い紫）でない限り、既婚のMiiはほぼ確実に断ります。通常は、別の相手を探すよう促すのが賢明です。

**Q: 親友なのに恋に落ちないのはなぜですか？**

A: 年齢設定を確認してください。一方が「子供」で他方が「大人」の場合、恋愛オプションはシステム的にロックされています。また、Miiの設定で恋愛対象の性別が正しく設定されているかも確認してください。
