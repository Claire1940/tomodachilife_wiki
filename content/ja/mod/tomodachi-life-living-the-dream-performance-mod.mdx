<Section icon={Target} title="PC およびコンソール用インストールガイド">

**トモダチコレクション 新生活 パフォーマンス MOD** のインストールは、選択したプラットフォームによって異なります。Ryujinx または Yuzu を使用する PC ユーザーは、改造されたニンテンドーSwitchで Atmosphere を実行するユーザーとは異なるワークフローになります。ファイルを正しいディレクトリに配置するために、以下の手順を注意深く実行してください。

<Steps>
  <Step num="1" title="正しいバージョンをダウンロードする">
    ゲームのバージョン (通常は 1.0.0 または 1.0.1) を確認します。GameBanana や GBAtemp などの信頼できるソースから、対応する 60 FPS の .7z ファイルまたはグラフィック MOD の .zip をダウンロードします。
  </Step>
  <Step num="2" title="エミュレータディレクトリのセットアップ (PC)">
    エミュレータでゲームを右クリックします。「Mod ディレクトリを開く」を選択します。Yuzu の場合は、ここにフォルダを抽出します。Ryujinx の場合は、`cheats` サブフォルダとビルド ID にちなんだ名前の `.txt` ファイルを作成する必要がある場合があります。
  </Step>
  <Step num="3" title="Atmosphere パッチ適用 (Switch)">
    `.ips` パッチを `sd:/atmosphere/exefs_patches/` ディレクトリに配置します。SD カードを整理するために、フォルダ名が分かりやすいもの (例: `TLLtD-Graphics`) であることを確認してください。
  </Step>
  <Step num="4" title="有効化と確認">
    エミュレータまたはコンソールを起動します。「チート管理」または「MOD マネージャー」メニューで、60 FPS またはグラフィック MOD がチェックされていることを確認します。変更を適用するためにゲームを再起動します。
  </Step>
</Steps>

<Section icon={Shield} title="最適化とトラブルシューティング">

**トモダチコレクション 新生活 パフォーマンス MOD** を使用すると、地面のテクスチャのちらつきや Mii アイコンのぼやけなど、視覚的なバグが発生することがあります。これらの問題は、エミュレータが異方性フィルタリングや LOD オフセットをどのように処理するかに関連していることがよくあります。「バグのある地面のテクスチャ」に遭遇した場合、それはグラフィック MOD と GPU ドライバー設定の間の競合である可能性が高いです。

| 問題 | 潜在的な原因 | 推奨される修正 |
| :--- | :--- | :--- |
| **地面のちらつき** | ドライバーの非互換性 | GPU ドライバーの更新 / Vulkan への切り替え |
| **Mii アイコンのぼやけ** | LOD オフセット MOD | グラフィック MOD を無効にする / x1 スケールを使用する |
| **オーディオの同期ずれ** | 60 FPS の同期ずれ | 専用のオーディオ同期パッチを使用する |
| **ゲームのクラッシュ** | 高 FPS 負荷 | FPS を 45 に制限するか、30 に戻す |

<Callout type="info" title="パフォーマンスのヒント">
Ryujinx では、「ゲストインターネットアクセスを有効にする」を有効にしてもパフォーマンスとは関係ありませんが、「グラフィックバックエンド」を Vulkan に設定すると、AMD および Intel ハードウェアでの 60 FPS MOD の安定性が大幅に向上します。
</Callout>

| グラフィック設定 | 推奨値 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| **解像度スケール** | 2倍 (1440p) | 中程度 |
| **異方性フィルタリング** | 4倍 または 16倍 | 低 |
| **V-Sync** | オフ (60 FPS MOD を使用している場合) | 高 |
| **シェーダーキャッシュ** | 有効にする | 高 (初回ロード時) |
