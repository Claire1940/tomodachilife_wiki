## 進化したMiiスタジオ：新しいカスタマイズオプション

『トモダチコレクション Living the Dream』のMiiスタジオは、前作よりも大幅に強化されています。Nintendo Switch版『ミートピア』の機能を継承しつつ、全く新しい要素を追加したことで、より表現豊かで多様なキャラクターを作成できるようになりました。最も注目すべき追加要素の一つは、これまでの標準的なMiiデザインにはなかった「人間のような耳」の導入です。

プレイヤーは髪型を「前髪」と「後ろ髪」に分けて設定できるようになり、数千通りの組み合わせが可能になりました。さらに、人間離れした肌の色、フェイスペイント、詳細なメイクアップオプションも導入されています。

| 機能 | 説明 | カスタマイズの深さ |
| :--- | :--- | :--- |
| **ヘアスタイル** | 前髪と後ろ髪を個別に設定可能。サブカラーやメッシュも。 | 高 |
| **顔の細部** | フェイスペイント、メイク、特定の耳の形を追加。 | 高 |
| **アイデンティティ設定** | 性別（男性、女性、ノンバイナリー）と代名詞を独立して設定。 | **インクルーシブ** |
| **恋愛の好み** | あらゆる性別の組み合わせ（または無し）への関心を選択可能。 | **インクルーシブ** |

> **💡 ヒント:** 「こだわり設定」メニューを使用して、Miiの動きや話し方をさらに細かく調整できます。食後に口を拭くかどうかまで設定可能です！

## トモダチコレクション Living the Dream Wiki：主要な場所と建物

3DS時代の固定されたマンションとは異なり、『Living the Dream』の島は高度にカスタマイズ可能です。建物を移動したり、地面のタイルを変更したり、島の形そのものを修正したりすることもできます。各施設は、Miiの幸福度を維持し、ゲームを進行させるための特定の目的を持っています。

| 建物名 | 機能 | 解放条件 |
| :--- | :--- | :--- |
| **フレッシュ・キングダム** | 日替わり商品が並ぶメインの食品スーパー。 | 最初から利用可能 |
| **ウェア＆ウェア** | 服、帽子、全身コスチュームを販売。 | 住人を2人追加 |
| **願いの泉** | **ウォーム・ファジー**を特別なレベルアップギフトと交換。 | 島のレベル1に到達 |
| **クイック・ビルド** | 島の装飾品、フェンス、植物を購入。 | 住人を5人追加 |
| **パレット・ハウス** | 服、食べ物、タイルのカスタムデザインを作成。 | 島のレベル3に到達 |
| **ニュースステーション** | 住人が登場する日替わりの「Miiニュース」を視聴。 | 最初から利用可能 |

### 願いの泉とウォーム・ファジー
**願いの泉**は、本作で極めて重要な新しい場所です。単なる経験値を得る代わりに、Miiとの交流によって「ウォーム・ファジー」と呼ばれる謎の感謝のオーブが発生します。これを集めることで島自体のレベルが上がり、新しい施設や住人のための「こだわり設定」が解放されます。

## ソーシャルメカニクスと人間関係の多様性

**トモダチコレクション Living the Dream Wiki**の議論の中で最も要望が多かった機能の一つが、同性同士の恋愛の導入でした。任天堂はこれに応え、性自認に関係なくMiiがロマンチックな絆を築けるようにしました。また、人間関係は完全にランダムではなくなり、「ハンドカーソル」を通じてプレイヤーがより積極的に介入できるようになっています。

### 手動でのインタラクション
ハンドカーソルを使用すると、物理的にMiiを持ち上げて他のMiiの近くに配置し、交流を促すことができます。この「おたすけハンド」メカニズムにより、誰が友達になるか、あるいは誰が恋心を抱くかに影響を与えることができます。しかし、シリーズ特有のカオスさは健在です。Miiには独自の意思があり、プレイヤーの仲介を拒否することもあります。

### 性格のアーキタイプ
基本的な性格診断グリッドは健在ですが、2026年の多様な行動をより反映するようにカテゴリー名が刷新されました。各性格タイプは、Miiのデフォルトの服や部屋のインテリアの特定のカラーパレットに対応しています。

*   **思いやり（黄）:** 思慮深く正直で、友人を第一に考える。
*   **野心家（紫/青）:** 自信家で結果重視。リーダーシップを発揮することが多い。
*   **独創的（その他）:** 予測不能でエネルギッシュ。風変わりな夢を見やすい。

> **⚠️ 警告:** 「野心家」のMiiを「独創的」なMiiに紹介する際は注意してください。親友になれる可能性もありますが、エネルギーの衝突により頻繁に「湯気が立つ」ほどの喧嘩に発展することがあります。

## 初代トモダチコレクションとの違い

3DSからNintendo Switchへの移行に伴い、技術面およびゲームプレイ面でいくつかの変更が行われました。食事を与え、服を着せ、Miiを観察するというコアなループは変わりませんが、シミュレーションの規模が変化しています。

| 機能 | トモダチコレクション (3DS) | Living the Dream (Switch) |
| :--- | :--- | :--- |
| **住居** | 全員が一つのマンションに居住。 | 各Miiがカスタマイズ可能な個別の一軒家に居住。 |
| **Miiの制限** | 最大100人。 | **70人に減少。** |
| **オンライン** | すれちがい通信、いつの間に間に通信。 | すれちがい通信なし。ローカル/オンライン共有に特化。 |
| **交流** | 主にメニューベース。 | 「ハンドカーソル」による物理的な介入。 |
| **ビジュアル** | デフォルメされた3Dモデル。 | 高精細テクスチャとアニメ風の赤面表現。 |

Miiの登録上限が100人から70人に減少したことは、コミュニティ内で議論の的となっています。しかし開発者は、最大8人まで住める拡張可能な一軒家など、Miiの住居の複雑さが増したことで、より多くのシステムリソースが必要になったと説明しています。

## 体験版（Welcome Edition）

本編を購入していない方のために、『トモダチコレクション Living the Dream - Welcome Edition（体験版）』が実質的なプレビューとして用意されています。2026年3月にリリースされたこのデモ版では、最大3人のMiiを作成し、衣料品店「ウェア＆ウェア」を解放するまでプレイできます。

**体験版完了特典:**
1.  **ハムスターの服:** 製品版に引き継ぎ可能なユニークな衣装。
2.  **セーブデータ引き継ぎ:** Miiの人間関係や「ウォーム・ファジー」を含むすべての進行状況が引き継がれます。
3.  **住人の進行:** デモ版ではMiiのレベルアップは止まりますが、交流は続き、製品版で登場する新機能について話すようになります。

デモ版を最大限に活用するには、データを引き継ぐ前に最初の3人のMiiの幸福度を最大化し、できるだけ多くのウォーム・ファジーを蓄積しておくことをお勧めします。リージョンごとのバージョンや言語サポートの詳細は、[任天堂公式サイト](https://www.nintendo.com)で確認できます。

## FAQ

**Q: 一軒の家に複数のMiiを住ませることはできますか？**

A: はい！『Living the Dream』では、Miiの家を拡張して最大8人まで一緒に住ませることができます。これは、結婚したカップルだけが一緒に住んでいた前作からの大きな変更点です。

**Q: パレット・ハウスはどうすれば解放されますか？**

A: パレット・ハウスは、島のレベルが3に達すると解放されます。食事を与えたり、悩みを解決したり、ミニゲームで遊んだりして、Miiからウォーム・ファジーを集めることで島のレベルを上げることができます。

**Q: 2026年版にリージョンごとの違いはありますか？**

A: はい、主にローカライズされたテキストの違いがあります。例えば、夢のシーンに登場する巨大なMiiは、北米（NTSC）版では「Did you want to talk?（話したかった？）」と言いますが、欧州（PAL）版では「You called?（呼んだ？）」と言います。

**Q: すれちがい通信機能はどうなりましたか？**

A: Nintendo Switchにはすれちがい通信のハードウェア機能がないため、これらの機能は削除されました。代わりに、ゲーム内のマーケットプレイスや、MiiのQRコードやカスタムデザインを共有するための強化されたオンラインツールに置き換わっています。
