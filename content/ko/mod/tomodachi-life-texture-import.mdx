| **텍스처가 게임에 나타나지 않음** | 잘못된 파일 형식, 잘못된 아이템 ID, 부적절한 저장 | PNG 형식 확인, 아이템 유형/ID 재확인, 툴 사용 전 게임 올바르게 저장 |
| **모딩 후 게임 충돌** | 손상된 세이브 파일, 호환되지 않는 모드, 잘못된 툴 사용 | 백업에서 복원, 툴 지침 확인, 다른 모드 시도 |
| **Mii 페이스 페인팅이 전송되지 않음** | *토모다치 라이프*의 한계, 툴 호환성 문제 | 페이스 페인팅이 덜 복잡한 Mii를 사용하거나 게임 내에서 수동으로 조정 |
| **백신 프로그램이 모딩 툴을 차단함** | 새롭게 독립적으로 컴파일된 `.exe` 파일 | 백신에서 예외로 추가하거나 "추가 정보"를 클릭하고 "실행" 선택 (일반적인 오진) |
| **모딩 툴이 UGC 폴더를 찾지 못함** | 잘못된 경로, 게임이 제대로 저장되지 않음 | `%appdata%` 경로 재확인, 게임 저장 및 종료 확인 |

<Callout type="warning" title="백신 경고">
새롭거나 독립적으로 개발된 많은 모딩 툴이 Windows Defender 또는 기타 백신 소프트웨어에서 "SmartScreen" 경고로 표시될 수 있습니다. 이는 종종 오진입니다. 소스를 신뢰한다면 일반적으로 "추가 정보"를 클릭하고 "실행"을 선택할 수 있습니다.
</Callout>
