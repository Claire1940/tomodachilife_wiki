### 권장 베이스라인 프로필

| 설정 영역 | 권장 값 | 이유 |
|---|---|---|
| **API/렌더러** | **Vulkan** | 많은 크래시 사례에서 더 나은 호환성 |
| **GPU 장치** | **고성능 전용 GPU** | 성능이 낮은 내장 그래픽이 우연히 선택되는 문제 방지 |

## 크래시 해결: 시작 시, 시작 후, 랜덤 프리징 시나리오

### B) 게임 플레이 진입 후 크래시
1. 파이프라인 캐시 정리
2. 캐시 저장소 정리

2. 전용 GPU가 선택되었는지 확인

4. 최근 변경한 설정 하나 되돌리기
5. 짧은 세션으로 테스트해 트리거 분리
