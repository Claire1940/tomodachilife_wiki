## Tomodachi Life Living the Dream ROM 업데이트: 설치 전 신뢰 체크리스트

파일을 건드리기 전에 이 체크리스트를 실행하세요. 10~15분이면 되고, 몇 시간짜리 복구 작업을 줄일 수 있습니다.

### 3) 파일 무결성 검증
체크섬(SHA-256/MD5)이 제공되면 다운로드 후 직접 대조하세요. 해시가 다르면 설치하지 마세요.

### 4) “사람 인증” 게이트 피하기
정상적인 패치 과정이라면 랜덤 앱 다운로드나 광고 작업 완료를 요구하지 않습니다.

### 5) 패치 파일과 실행 파일 구분하기
ROM 관련 업데이트 대부분은 설치 EXE가 아니라 데이터 패치입니다. 출처 불명의 실행 파일은 특히 주의하세요.

| **신원** | 관리자에게 업데이트 이력이 있음 | 이력이 전혀 없는 신규 익명 업로드 |
| **문서화** | 명확한 패치 방식과 버전 매핑 | 기술 설명 없이 “이 설치기만 실행” |

| **배포** | 커뮤니티 미러 + 토론 스레드 | 긴급함만 강조하는 단축 링크 페이지 하나 |
