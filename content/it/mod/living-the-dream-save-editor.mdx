## Preparazione alla Modifica: Posizione dei File

Prima di poter utilizzare il **living the dream save editor**, devi essere in grado di individuare e fare il backup dei tuoi file di salvataggio. A seconda che tu stia utilizzando l'[emulatore Ryujinx](https://ryujinx.org/) o una console Nintendo Switch fisica con firmware personalizzato, il processo per trovare questi file differisce leggermente.

### Trovare i Dati di Salvataggio su Ryujinx

Per la maggior parte dei giocatori su PC, Ryujinx è il metodo principale per vivere questo progetto. L'emulatore memorizza i dati di salvataggio in una directory nidificata all'interno della cartella AppData del sistema.

1.  Premi `Tasto Windows + R` e digita `%appdata%`.
2.  Naviga in `Ryujinx` > `bis` (o `bin`) > `user` > `save`.
3.  Vedrai diverse cartelle con lunghe stringhe numeriche. Dovrai cercarle finché non trovi quella contenente `player.save`.
4.  **Importante:** Crea sempre una copia di backup del tuo `player.save` prima di procedere con qualsiasi modifica.

### Trovare i Dati di Salvataggio su una Switch Modificata

Se stai giocando su hardware reale, avrai bisogno di un gestore di salvataggi come JKSV o Checkpoint.

*   Esporta i dati di salvataggio per "Tomodachi Life: Living the Dream" utilizzando il tuo gestore preferito.
*   Collega la tua scheda SD al PC o usa un client FTP per accedere alla cartella esportata.
*   Cerca il file `player.save` all'interno della directory esportata.

> **Attenzione:** La modifica dei file di salvataggio su una console fisica comporta un piccolo rischio di corruzione dei dati. Assicurati che la console sia in Modalità Aereo o bloccata dai server ufficiali per prevenire problemi di sincronizzazione.

## Risoluzione dei Problemi Comuni

Sebbene il **living the dream save editor** sia molto affidabile, gli utenti potrebbero incontrare piccoli ostacoli durante il primo tentativo di modding.

### Il File di Salvataggio non si Carica
Se Ryujinx non si avvia dopo aver sostituito il file, potresti averlo rinominato accidentalmente. Assicurati che il nome del file sia esattamente `player.save` (tutto minuscolo) e che sia posizionato nella corretta sottocartella numerica.

### I Mii Menzionano Ancora la Versione Retail
Questo accade se il "Lockdown Status" non è stato attivato correttamente o se non hai sovrascritto il file giusto. Esegui nuovamente l'editor e verifica che lo "Status Lockdown" sia impostato su "Disabled" prima di scaricare il nuovo salvataggio.

### Felicità e Livellamento
Nella "Welcome Version", gli sviluppatori potrebbero aver limitato la barra della felicità. Sebbene il **living the dream save editor** permetta ai Mii di interagire, potrebbero non guadagnare sempre livelli o "Warm Fuzzies" allo stesso modo della versione retail completa. Questo è un limite del motore della demo stessa piuttosto che un difetto dell'editor.
