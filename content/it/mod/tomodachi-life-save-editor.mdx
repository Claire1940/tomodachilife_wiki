## Passaggio 1: Esportazione dei Dati di Salvataggio con JKSM

Il primo ostacolo è estrarre il file di salvataggio dal 3DS e portarlo sul computer. Il gioco memorizza i suoi dati in un formato che non è immediatamente visibile se si inserisce semplicemente la scheda SD in un PC. È necessario utilizzare uno strumento come **JK Save Manager (JKSM)** per "estrarre" i dati di salvataggio in un formato leggibile.

1.  Avvia **JKSM** sul tuo 3DS con homebrew.
2.  Naviga nel menu "Titles" e individua **Tomodachi Life** nella lista dei giochi installati.
3.  Seleziona "Save Data" e poi scegli "Export Save".
4.  Crea una nuova cartella per l'esportazione. È buona norma nominare questa cartella con la data corrente (ad esempio, `2026-03-30`) per tenere traccia dei progressi.
5.  Una volta completato il processo, JKSM avrà creato un file chiamato `savedata.arc` all'interno della cartella sulla tua scheda SD.

Questo file `savedata.arc` è il "cervello" della tua isola. Contiene tutto, dal colore dei calzini dei tuoi Mii al saldo attuale del conto bancario dell'isola.

## Passaggio 4: Ripristino del Salvataggio sul 3DS

Dopo aver apportato le modifiche desiderate nel **tomodachi life save editor**, devi cliccare su "Save" all'interno dell'applicazione per sovrascrivere il file `savedata.arc` sulla tua scheda SD. Quindi, segui questi passaggi per riportare le modifiche sulla tua console:

1.  Espelli in sicurezza la scheda SD dal PC e inseriscila nuovamente nel tuo 3DS.
2.  Apri di nuovo **JKSM**.
3.  Torna su **Tomodachi Life** > "Save Data".
4.  Evidenzia la cartella che hai creato in precedenza (quella contenente il file modificato).
5.  Premi il pulsante per "Restore" (solitamente il tasto 'Y' o 'A' a seconda della versione).
6.  Conferma il ripristino. Questo sovrascriverà i dati di gioco correnti con la tua versione modificata.

Avvia Tomodachi Life. Se eseguito correttamente, dovresti vedere immediatamente i tuoi totali di denaro modificati, gli oggetti sbloccati o le relazioni aggiornate.

## Risoluzione dei Problemi Comuni

Sebbene l'editing dei salvataggi sia generalmente semplice, potresti riscontrare occasionali intoppi. La maggior parte dei problemi deriva da errori nei percorsi dei file o discrepanze di regione.

| Problema | Possibile Causa | Soluzione |
| :--- | :--- | :--- |
| **Salvataggio non caricato** | Corruzione o regione errata | Assicurati che l'editor sia impostato sulla regione del tuo gioco (USA/EUR/JPN). |
| **JKSM non vede il gioco** | Title DB non aggiornato | Aggiorna la lista dei titoli nelle impostazioni di JKSM. |
| **Le modifiche non appaiono** | Passaggio Restore saltato | Assicurati di aver premuto "Restore" in JKSM, non solo "Export". |
| **L'editor non si apre** | .NET Framework mancante | Aggiorna le dipendenze di Windows per eseguire il file `.exe`. |

Per garantire i migliori risultati, usa sempre una [scheda SD ufficiale Nintendo](https://www.nintendo.com/) o un ricambio di alta qualità, poiché le schede SD economiche sono inclini alla corruzione dei dati durante il processo di iniezione del salvataggio.
