<Section icon={Map} title="Installazione e Configurazione Iniziale">

La configurazione dell'emulatore richiede precisione, specialmente per quanto riguarda i file di sistema. Se le chiavi o il firmware non corrispondono, il gioco non si avvierà o mostrerà un errore "Keys not found". Segui questi passaggi per garantire un'installazione pulita.

<Steps>
  <Step num="1" title="Scarica Ryujinx Canary">
    Visita il sito ufficiale o la pagina dei rilasci **tomodachi life github** per scaricare la build Ryujinx Canary. Estrai la cartella "publish" sul tuo desktop e rinominala per un facile accesso. Esegui l'eseguibile e ignora l'avviso iniziale "Keys not found".
  </Step>
  <Step num="2" title="Installa Prod e Title Keys">
    Naviga su `File > Open Ryujinx Folder`. Vai nella directory `system`. Incolla qui i tuoi file `prod.keys` e `title.keys`. Riavvia l'emulatore per applicare le modifiche; il messaggio di errore dovrebbe scomparire.
  </Step>
  <Step num="3" title="Flash del Firmware di Sistema">
    Vai su `Tools > Install Firmware`. Seleziona "Install from XCI or ZIP" e scegli il tuo file firmware (es. v20.5.0). Conferma l'installazione. Ryujinx mostrerà ora la versione del sistema nell'angolo in basso a destra.
  </Step>
  <Step num="4" title="Mappa le Directory di Gioco">
    In `Options > Settings`, vai alla scheda "Game Directories". Clicca su "Add" e seleziona la cartella dove sono archiviati i tuoi file XCI o NSP di Tomodachi Life. Fai lo stesso per la cartella "Updates and DLC" se applicabile.
  </Step>
</Steps>

<Callout type="tip" title="Consiglio Pro">
Se stai usando un laptop con doppia scheda grafica, assicurati che Ryujinx sia impostato per utilizzare la tua GPU "High Performance" (NVIDIA/AMD) nelle impostazioni di visualizzazione di Windows, piuttosto che la grafica integrata Intel/AMD.
</Callout>

</Section>
