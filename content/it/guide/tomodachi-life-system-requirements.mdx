### Priorità principali della configurazione

1. Usa una build stabile e aggiornata dell’emulatore.
2. Imposta la velocità di emulazione al 100% prima di toccare le opzioni turbo.
3. Seleziona la GPU dedicata corretta nelle impostazioni dell’emulatore.
4. Inizia con risoluzione interna 2x o 3x.
5. Aumenta la risoluzione solo dopo una sessione di test completa.

| Area impostazioni | Valore iniziale sicuro | Quando cambiarlo |
|---|---|---|
| **Risoluzione interna** | **3x nativa** | Aumenta gradualmente se il frametime è stabile |
| **API grafica** | **Prima OpenGL** | Prova Vulkan se OpenGL ha scatti |

## Problemi comuni e soluzioni (inclusi bug legati ai Mii)

Anche se il tuo PC soddisfa i **tomodachi life system requirements**, potresti incontrare problemi noti dell’emulazione. Ecco soluzioni pratiche.

| Problema | Causa probabile | Soluzione rapida |
|---|---|---|
| **Mii invisibili o senza testa** | File di compatibilità mancanti o dati di gioco corrotti | Applica il metodo Mii-fix noto per la tua versione |
| **Scatti casuali dopo lunghe sessioni** | Cache shader gonfiata | Pulisci la cache shader OpenGL/Vulkan |
| **Audio con crepitii** | Incompatibilità del backend audio | Cambia modalità output (Auto → SDL2/OpenAL) |
| **Crash all’avvio** | Impostazioni corrotte/mismatch aggiornamento | Reimposta la configurazione e riapplica gradualmente le impostazioni |
| **Controller non risponde** | Mappatura profilo errata | Rimappa ogni input e salva un profilo dedicato |

> **Suggerimento:** Cambia una variabile alla volta. Se modifichi insieme API, risoluzione e backend audio, non saprai quale modifica ha davvero risolto il problema.

### Checklist per archiviazione e gestione file

Molti errori di avvio sono dovuti a formati file sbagliati, non a limiti hardware. Tieni tutto ordinato:

- Estrai completamente gli archivi compressi prima dell’uso

- Evita directory annidate troppo in profondità
- Usa un’unità SSD per un comportamento di caricamento migliore
- Fai backup dei salvataggi prima di cambi di configurazione importanti
