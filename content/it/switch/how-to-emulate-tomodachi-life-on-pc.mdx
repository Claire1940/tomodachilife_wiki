## Come Emulare Tomodachi Life su PC (Configurazione Passo Passo)

Questo è il flusso completo di installazione da seguire nel 2026.

1. **Scarica Ryujinx Canary**
   - Usa il ramo Canary, non le build legacy/solo stabili, se le note di compatibilità per la tua versione target lo richiedono.
   - Estrai l’archivio in una cartella pulita (ad esempio: `D:\Emulation\Ryujinx`).

2. **Avvia Ryujinx una volta**
   - Il primo avvio crea le cartelle necessarie.
   - Se compare Windows SmartScreen, conferma e continua.

3. **Aggiungi le chiavi**
   - In Ryujinx, apri la directory app data/system.
   - Inserisci `prod.keys` e `title.keys` nella cartella `system`.
   - Riavvia Ryujinx.

4. **Installa il firmware**
   - Usa **Tools/Actions → Install Firmware from ZIP** (la dicitura può variare in base alla build).
   - Lascia completare l’installazione, poi riavvia di nuovo.

5. **Imposta le directory dei giochi**
   - Apri **Options → Settings → Game Directories**.
   - Aggiungi la cartella del gioco che contiene XCI/NSP.
   - Se tieni separatamente le cartelle update/DLC, aggiungi anche quelle.

6. **Mappa i comandi del controller**
   - Vai nelle impostazioni **Input**, scegli il pad collegato e assegna un profilo.
   - Se la disposizione dei pulsanti non ti convince, rimappa ABXY manualmente.

7. **Applica impostazioni grafiche di base**
   - Inizia con Vulkan e con la tua GPU dedicata selezionata.
   - Mantieni prima la risoluzione nativa, poi passa all’upscale solo dopo aver verificato la stabilità.
