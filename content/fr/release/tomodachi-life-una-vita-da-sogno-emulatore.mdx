<Section icon={Cpu} title="Configuration de l'Émulateur Étape par Étape">

La configuration de votre **émulateur tomodachi life una vita da sogno** nécessite quelques fichiers spécifiques pour contourner les couches de sécurité de Nintendo. Suivez ces étapes pour faire fonctionner votre île en quelques minutes.

<Steps>
  <Step num="1" title="Télécharger et Installer l'Émulateur">
    Visitez le GitHub officiel ou le site web de Ryujinx. Téléchargez la dernière version optimisée "Living the Dream". Extrayez les fichiers dans un dossier dédié comme `C:\Games\TomodachiEmulator`.
  </Step>
  <Step num="2" title="Installer le Firmware et les Clés">
    Ouvrez l'émulateur et naviguez vers le dossier Système. Vous devez placer vos `prod.keys` et `title.keys` dans le répertoire `/system`. Ensuite, allez dans `Outils > Installer le Firmware` et sélectionnez votre fichier `.zip` ou `.xci` de firmware.
  </Step>
  <Step num="3" title="Configurer les Paramètres Graphiques">
    Allez dans `Options > Paramètres > Graphiques`. Définissez le Backend sur **Vulkan**. Si vous avez une carte NVIDIA, activez **NVIDIA Reflex**. Définissez l'échelle de résolution sur **2x (1440p)** ou **3x (4K)** pour un aspect net.
  </Step>
  <Step num="4" title="Initialiser le Jeu et la Démo">
    Ajoutez votre répertoire de jeu à la liste. Cliquez droit sur *Tomodachi Life: Living the Dream* et sélectionnez "Gérer les Mises à Jour du Titre" pour appliquer le patch Day-One 2026. Lancez le jeu pour commencer le processus de création de l'île.
  </Step>
</Steps>
