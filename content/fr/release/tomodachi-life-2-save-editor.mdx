## Prérequis pour l'édition de sauvegarde

Avant de plonger dans le monde de la manipulation de sauvegardes, vous devez comprendre qu'un **éditeur de sauvegarde Tomodachi Life 2** ne modifie pas le code du jeu ; il édite plutôt les valeurs spécifiques stockées dans votre fichier de sauvegarde. Pour accéder à ces fichiers sur une Nintendo 3DS, votre console doit être équipée d'un Custom Firmware (CFW).

Le processus de "hack" ou de "modding" de votre 3DS est plus accessible en 2026 que jamais, mais il nécessite un matériel spécifique. Assurez-vous d'avoir un ordinateur avec un lecteur de carte SD et une carte SD d'une capacité comprise entre 2 Go et 128 Go.

| Exigence | Spécification | Objectif |
|---|---|---|
| **Console** | N'importe quel modèle 3DS/2DS | La plateforme pour le jeu. |
| **Firmware** | Luma3DS (CFW) | Permet l'accès aux fichiers système. |
| **Carte SD** | 2 Go - 128 Go (FAT32) | Stocke les mods, les sauvegardes et les données. |
| **Gestionnaire de sauvegarde** | Checkpoint | Sauvegarde et restaure vos fichiers de sauvegarde. |
| **Outil d'édition** | Save Editor v4.0.1 | Le logiciel utilisé pour modifier les valeurs. |

> **⚠️ Avertissement :** Créez toujours une sauvegarde "NAND" avant de modder votre système. Il s'agit d'un instantané complet des fichiers système de votre 3DS qui peut être utilisé pour "débriquer" votre console si quelque chose tourne mal pendant l'installation du CFW.

## Étape par étape : Comment utiliser l'éditeur de sauvegarde

Une fois votre 3DS moddée et Checkpoint installé, vous êtes prêt à commencer. L'**éditeur de sauvegarde Tomodachi Life 2** fonctionne en récupérant une sauvegarde créée par Checkpoint et en l'ouvrant sur votre PC.

1.  **Créer une sauvegarde :** Insérez votre cartouche Tomodachi Life (ou ouvrez la version numérique), lancez Checkpoint, sélectionnez le jeu et appuyez sur "New" pour créer une sauvegarde.
2.  **Transférer sur PC :** Éteignez votre 3DS, insérez la carte SD dans votre ordinateur et naviguez vers `/3DS/Checkpoint/saves/Tomodachi Life/[Nom de votre sauvegarde]`.
3.  **Ouvrir l'éditeur :** Lancez le Tomodachi Life Save Editor et sélectionnez le fichier `savedata.arc` sur votre carte SD.
4.  **Modifier les valeurs :** Utilisez les différents onglets (Island, Mii, Relationships, Items) pour effectuer les changements souhaités.
5.  **Enregistrer et restaurer :** Cliquez sur "Save" dans l'éditeur, remettez la carte SD dans votre 3DS, et utilisez Checkpoint pour "Restore" la sauvegarde modifiée.

## Fonctionnalités clés de l'éditeur de sauvegarde

La beauté de l'utilisation d'un **éditeur de sauvegarde Tomodachi Life 2** réside dans la finesse du contrôle. Vous ne changez pas seulement des chiffres ; vous réécrivez l'histoire de votre île.

### Personnalisation de l'île et des Miis
Dans le menu Island Edit, vous pouvez débloquer instantanément tous les lieux, tels que le parc d'attractions ou le café, sans attendre les déclencheurs de progression naturelle. Plus important encore, le menu Mii Edit vous permet de changer le nom d'un Mii, ses aliments préférés et même son année de naissance.

### Le panneau des relations (Relationships)
C'est la section la plus populaire de l'éditeur. Vous pouvez forcer deux Miis à devenir meilleurs amis, époux ou ennemis jurés. Cependant, vous devez être conscient de l'erreur "No Correspondence". Cela se produit lorsque l'éditeur détecte un état de relation qu'il ne reconnaît pas, généralement des événements rares comme un Mii voulant se remettre avec un ex.

| Action de relation | Niveau de risque | Résultat |
|---|---|---|
| **Augmenter l'amitié** | Faible | Les Miis deviennent instantanément de meilleurs amis. |
| **Forcer le mariage** | Moyen | Les Miis emménagent dans une maison ; nécessite un emplacement vide. |
| **Mariage homosexuel** | **Élevé** | Provoque souvent des plantages immédiats ou une corruption de sauvegarde. |
| **Augmenter le niveau du Mii** | Faible | Débloque immédiatement des emplacements de cadeaux et des chansons. |

## Éviter la corruption de sauvegarde : Le bug "Friend All"

Bien qu'un **éditeur de sauvegarde Tomodachi Life 2** soit incroyablement utile, il peut être dangereux s'il est utilisé de manière imprudente. La cause la plus fréquente d'un fichier de sauvegarde corrompu est le bouton "Friend All" (Tout ami) présent dans de nombreuses versions de l'outil.

Le jeu suit les Miis en utilisant des emplacements numérotés de 1 à 100. Lorsque vous supprimez un Mii, cet emplacement reste vide jusqu'à ce qu'un nouveau Mii soit créé. Si vous utilisez la fonction "Friend All", l'éditeur peut accidentellement attribuer le statut d'ami à un emplacement vide. Lorsque vous essaierez plus tard de créer un nouveau Mii dans cet emplacement, le jeu rencontrera un conflit : il verra un emplacement vide qui possède déjà une liste d'amis. Cela crée ce que la communauté appelle le "Mii de Schrödinger" — un personnage qui existe et n'existe pas à la fois — entraînant un plantage permanent lors de l'entrée dans les zones de la fontaine ou des appartements.

> **💡 Astuce de pro :** Évitez les boutons de modification "en masse". Il est toujours plus sûr de modifier les relations des Miis individuellement pour s'assurer qu'aucun emplacement vide n'est modifié par erreur.

## Gestion des objets et de l'inventaire

Pour les collectionneurs, l'éditeur offre un moyen de contourner les rotations quotidiennes des boutiques. Vous pouvez vous donner 99 exemplaires de chaque aliment, vêtement et style d'intérieur. C'est particulièrement utile pour obtenir les objets "Spéciaux" qui étaient initialement distribués via SpotPass, lequel n'est plus actif en 2026.

Si vous jouez à une version du jeu provenant d'une région différente (par exemple, une cartouche européenne sur une console américaine), vous pourriez rencontrer des erreurs. Vous pouvez utiliser le [Luma Locale Switcher](https://github.com/NightYoshi370/Luma-Locale-Switcher) pour tromper le jeu en lui faisant croire qu'il fonctionne sur son matériel d'origine, garantissant ainsi que les modifications de l'éditeur de sauvegarde sont lues correctement.

## FAQ

**Q : Puis-je utiliser l'éditeur de sauvegarde Tomodachi Life 2 sur un émulateur mobile comme Citra ?**

R : Oui. Étant donné que Citra émule le système de fichiers de la 3DS, il vous suffit de localiser le fichier `savedata.arc` dans la structure des dossiers de votre émulateur (généralement sous `AppData/Roaming/Citra/sdmc/...`) et de l'ouvrir avec l'éditeur.

**Q : Pourquoi le statut "Amoureux" de mon Mii disparaît-il le lendemain ?**

R : Il s'agit d'une limitation connue de la version 4.0.1. Si vous réglez un Mii sur "Amoureux" mais que vous n'interagissez pas avec l'icône du cœur en jeu avant la réinitialisation quotidienne, la logique interne du jeu peut écraser la modification de l'éditeur. Il est préférable d'éditer la sauvegarde, puis de lancer immédiatement le jeu pour déclencher la déclaration.

**Q : Y a-t-il un risque d'être banni du Nintendo Network pour avoir utilisé un éditeur de sauvegarde ?**

R : En 2026, l'attention de Nintendo sur les serveurs 3DS est minimale. Cependant, l'utilisation de Miis modifiés dans les fonctionnalités de partage en ligne (comme l'échange de codes QR Mii) pourrait théoriquement entraîner un signalement. Pour une utilisation locale en solo, il n'y a pratiquement aucun risque de bannissement de la console.

**Q : Puis-je utiliser cela pour créer des mariages homosexuels dans Tomodachi Life ?**

R : Bien que vous puissiez techniquement changer le sexe d'un Mii dans une relation à l'aide de l'éditeur, le code du jeu n'est pas conçu pour gérer cela. Cela entraîne fréquemment une corruption de sauvegarde ou un plantage du jeu lorsque les deux Miis interagissent. Il est fortement recommandé de conserver des sauvegardes avant de tenter l'expérience.
