## Comprendre le Living the Dream Save Editor

Le **living the dream save editor**, plus précisément l'outil développé par le membre de la communauté creeperyano99, est un utilitaire Web conçu pour modifier le fichier `player.save` présent dans la démo de Tomodachi Life: Living the Dream. Dans l'expérience standard de la démo, le jeu déclenche un statut de « verrouillage » (lockdown) une fois que vous avez créé trois Miis et ouvert le magasin de vêtements. À partir de ce moment, vos Miis cessent d'interagir avec le monde et ne parlent que de la sortie prochaine de la version complète.

En utilisant l'éditeur, vous pouvez basculer le « Statut de verrouillage » sur désactivé. Ce simple changement permet à vos Miis de quitter leur maison, d'interagir les uns avec les autres, de développer des amitiés et de vous présenter des problèmes à résoudre — transformant ainsi la démo en une version fonctionnelle du jeu.

## Préparation de la modification : Emplacement des fichiers

Avant de pouvoir utiliser le **living the dream save editor**, vous devez être en mesure de localiser et de sauvegarder vos fichiers de sauvegarde. Selon que vous utilisez l' [émulateur Ryujinx](https://ryujinx.org/) ou une console Nintendo Switch physique avec un firmware personnalisé, le processus pour trouver ces fichiers diffère légèrement.

### Trouver les données de sauvegarde sur Ryujinx

Pour la plupart des joueurs sur PC, Ryujinx est la méthode principale pour découvrir ce projet. L'émulateur stocke les données de sauvegarde dans un répertoire imbriqué dans le dossier AppData de votre système.

1.  Appuyez sur la `Touche Windows + R` et tapez `%appdata%`.
2.  Naviguez vers `Ryujinx` > `bis` (ou `bin`) > `user` > `save`.
3.  Vous verrez plusieurs dossiers avec de longues chaînes numériques. Vous devez les parcourir jusqu'à trouver celui contenant `player.save`.
4.  **Important :** Créez toujours une copie de sauvegarde de votre `player.save` avant de procéder à des modifications.

### Trouver les données de sauvegarde sur une Switch modifiée

Si vous jouez sur console physique, vous aurez besoin d'un gestionnaire de sauvegarde tel que JKSV ou Checkpoint.

*   Exportez les données de sauvegarde pour « Tomodachi Life: Living the Dream » en utilisant votre gestionnaire préféré.
*   Connectez votre carte SD à votre PC ou utilisez un client FTP pour accéder au dossier exporté.
*   Cherchez le fichier `player.save` dans le répertoire exporté.

> **Avertissement :** La modification des fichiers de sauvegarde sur une console physique comporte un léger risque de corruption de données. Assurez-vous que votre console est en mode avion ou bloquée des serveurs officiels pour éviter tout problème de synchronisation.

Localisez le commutateur **Lockdown Status** (Statut de verrouillage) et assurez-vous qu'il est réglé sur « Disabled » (Désactivé). Vous pouvez maintenant définir votre argent sur une valeur élevée. Bien que l'outil permette des sommes astronomiques, il est recommandé de rester dans la plage de 1 000 000 pour éviter les bugs d'interface. Ensuite, cliquez sur le bouton **Unlock Items** (Débloquer les objets). Cela remplira vos magasins de vêtements et de nourriture avec chaque ressource actuellement incluse dans la version démo, y compris les tenues qui sont habituellement verrouillées derrière des rotations quotidiennes.

## Résolution des problèmes courants

Bien que le **living the dream save editor** soit très fiable, les utilisateurs peuvent rencontrer des obstacles mineurs lors de leur première tentative de modding.

### Le fichier de sauvegarde ne se charge pas
Si Ryujinx ne parvient pas à démarrer après avoir remplacé le fichier, vous avez peut-être renommé le fichier par accident. Assurez-vous que le nom du fichier est exactement `player.save` (tout en minuscules) et qu'il est placé dans le bon sous-dossier numérique.

### Les Miis mentionnent toujours la version commerciale
Cela se produit si le « Lockdown Status » n'a pas été correctement basculé ou si vous n'avez pas écrasé le bon fichier. Relancez l'éditeur et vérifiez que le « Status Lockdown » est réglé sur « Disabled » avant de télécharger la nouvelle sauvegarde.

### Bonheur et niveaux
Dans la « Version de bienvenue », les développeurs ont peut-être plafonné la barre de bonheur. Bien que le **living the dream save editor** permette aux Miis d'interagir, ils ne gagneront peut-être pas toujours des niveaux ou des « petits bonheurs » de la même manière que dans la version commerciale complète. Il s'agit d'une limitation du moteur de la démo elle-même plutôt que d'un défaut de l'éditeur.

## FAQ

**Q : Puis-je utiliser le living the dream save editor sur la version 3DS originale de Tomodachi Life ?**

R : Non, cet éditeur spécifique est conçu pour le projet de fan « Living the Dream » et sa démo pour Nintendo Switch/Ryujinx. Pour la version 3DS, vous auriez besoin d'un outil différent spécifiquement conçu pour les fichiers `.sav` de cette console.

**Q : Est-il possible d'ajouter plus de trois Miis en utilisant cet outil ?**

R : La version actuelle de l'éditeur se concentre sur les statistiques du joueur, l'argent et les objets. Bien que la désactivation du verrouillage permette à vos Miis existants de se déplacer librement, la possibilité d'ajouter plus de Miis est souvent limitée par le code interne de la démo. Cependant, certains utilisateurs ont réussi en créant des Miis avant que le verrouillage ne se déclenche.

**Q : L'utilisation du living the dream save editor peut-elle faire bannir mon compte Nintendo ?**

R : Si vous utilisez un émulateur comme Ryujinx, il n'y a aucun risque pour votre compte Nintendo officiel. Si vous utilisez une Switch modifiée, vous devriez toujours rester hors ligne lorsque vous utilisez des données de sauvegarde modifiées pour éviter d'être détecté par les services de télémétrie de Nintendo.

**Q : L'éditeur fonctionne-t-il sur Mac ou Linux ?**

R : Oui. Comme l'éditeur est un outil Web fourni par la communauté, il fonctionne dans n'importe quel navigateur moderne, quel que soit votre système d'exploitation. Vous avez seulement besoin de pouvoir accéder à votre système de fichiers pour télécharger et importer le fichier `player.save`.
