<Section icon={Map} title="Installation et configuration initiale">

La configuration de l'émulateur nécessite de la précision, en particulier concernant les fichiers système. Si les clés ou le firmware ne correspondent pas, le jeu ne démarrera pas ou affichera une erreur "Clés introuvables". Suivez ces étapes pour garantir une installation propre.

<Steps>
  <Step num="1" title="Télécharger Ryujinx Canary">
    Visitez le site officiel ou la page des versions **tomodachi life github** pour télécharger la version Ryujinx Canary. Extrayez le dossier "publish" sur votre bureau et renommez-le pour un accès facile. Exécutez l'exécutable et ignorez l'avertissement initial "Clés introuvables".
  </Step>
  <Step num="2" title="Installer les clés de production et de titre">
    Accédez à `Fichier > Ouvrir le dossier Ryujinx`. Allez dans le répertoire `system`. Collez vos fichiers `prod.keys` et `title.keys` ici. Redémarrez l'émulateur pour appliquer les modifications ; le message d'erreur devrait disparaître.
  </Step>
  <Step num="3" title="Flasher le firmware système">
    Allez dans `Outils > Installer le firmware`. Sélectionnez "Installer depuis XCI ou ZIP" et choisissez votre fichier firmware (par exemple, v20.5.0). Confirmez l'installation. Ryujinx affichera maintenant la version du système dans le coin inférieur droit.
  </Step>
  <Step num="4" title="Mapper les répertoires de jeux">
    Dans `Options > Paramètres`, allez dans l'onglet "Répertoires de jeux". Cliquez sur "Ajouter" et sélectionnez le dossier où sont stockés vos fichiers XCI ou NSP de Tomodachi Life. Faites de même pour le dossier "Mises à jour et DLC" le cas échéant.
  </Step>
</Steps>

<Callout type="tip" title="Conseil de pro">
Si vous utilisez un ordinateur portable avec double carte graphique, assurez-vous que Ryujinx est configuré pour utiliser votre GPU "Haute performance" (NVIDIA/AMD) dans les paramètres d'affichage de Windows plutôt que la carte graphique intégrée Intel/AMD.
</Callout>
