## Étape 1 : Exporter vos données de sauvegarde avec JKSM

Le premier obstacle consiste à extraire votre fichier de sauvegarde de la 3DS pour le mettre sur votre ordinateur. Le jeu stocke ses données dans un format qui n'est pas immédiatement visible si vous branchez simplement votre carte SD sur un PC. Vous devez utiliser un outil comme **JK Save Manager (JKSM)** pour extraire les données de sauvegarde dans un format lisible.

1.  Lancez **JKSM** sur votre 3DS modifiée.
2.  Naviguez dans le menu « Titles » et localisez **Tomodachi Life** dans votre liste de jeux installés.
3.  Sélectionnez « Save Data » puis choisissez « Export Save ».
4.  Créez un nouveau dossier pour votre exportation. Il est conseillé de nommer ce dossier avec la date actuelle (ex: `2026-03-30`) pour suivre votre progression.
5.  Une fois le processus terminé, JKSM aura créé un fichier nommé `savedata.arc` à l'intérieur du dossier sur votre carte SD.

Ce fichier `savedata.arc` est le « cerveau » de votre île. Il contient tout, de la couleur des chaussettes de vos Miis au solde actuel du compte bancaire de votre île.

## Étape 4 : Restaurer la sauvegarde sur votre 3DS

Après avoir effectué les modifications souhaitées dans le **tomodachi life save editor**, vous devez cliquer sur « Save » dans l'application pour écraser le fichier `savedata.arc` sur votre carte SD. Ensuite, suivez ces étapes pour ramener les modifications sur votre console :

1.  Éjectez en toute sécurité la carte SD de votre PC et réinsérez-la dans votre 3DS.
2.  Ouvrez à nouveau **JKSM**.
3.  Naviguez de nouveau vers **Tomodachi Life** > « Save Data ».
4.  Mettez en surbrillance le dossier que vous avez créé précédemment (celui contenant votre fichier modifié).
5.  Appuyez sur le bouton pour « Restore » (généralement le bouton 'Y' ou 'A' selon votre version).
6.  Confirmez la restauration. Cela écrasera les données actuelles du jeu par votre version éditée.

Lancez Tomodachi Life. Si l'opération a été effectuée correctement, vous devriez voir vos totaux d'argent modifiés, vos objets débloqués ou vos relations mises à jour immédiatement.

## Dépannage des problèmes courants

Bien que l'édition de sauvegardes soit généralement simple, vous pouvez rencontrer des obstacles occasionnels. La plupart des problèmes proviennent d'erreurs de chemin de fichier ou d'incompatibilités de région.

| Problème | Cause potentielle | Solution |
| :--- | :--- | :--- |
| **La sauvegarde ne charge pas** | Corruption ou mauvaise région | Assurez-vous que l'éditeur est réglé sur la région de votre jeu (USA/EUR/JPN). |
| **JKSM ne voit pas le jeu** | Base de données de titres non mise à jour | Actualisez la liste des titres dans les paramètres de JKSM. |
| **Les changements n'apparaissent pas** | Étape de restauration oubliée | Assurez-vous d'avoir appuyé sur « Restore » dans JKSM, et pas seulement « Export ». |
| **L'éditeur ne s'ouvre pas** | .NET Framework manquant | Mettez à jour vos dépendances Windows pour exécuter le fichier `.exe`. |

Pour garantir les meilleurs résultats, utilisez toujours une [carte SD officielle Nintendo](https://www.nintendo.com/) ou un remplacement de haute qualité, car les cartes SD bon marché sont sujettes à la corruption des données pendant le processus d'injection de sauvegarde.

## FAQ

**Q : Puis-je utiliser un éditeur de sauvegarde Tomodachi Life sur un émulateur comme Citra ?**

R : Oui ! En fait, c'est même plus facile sur un émulateur. Il vous suffit de localiser le dossier « C-user » où Citra stocke les données de sauvegarde, de trouver le fichier `savedata.arc` et de l'ouvrir directement avec l'éditeur. Vous n'avez pas besoin de JKSM pour ce processus.

**Q : L'utilisation d'un éditeur de sauvegarde peut-elle entraîner le bannissement de mon Nintendo Network ID ?**

R : C'est très peu probable. Comme Tomodachi Life est principalement un jeu hors ligne et que les serveurs 3DS sont largement abandonnés par Nintendo en 2026, il n'y a pas de surveillance active des modifications de sauvegardes. Cependant, utilisez toujours le homebrew à vos propres risques.

**Q : Puis-je changer le type de personnalité d'un Mii avec l'éditeur ?**

R : Oui. L'éditeur vous permet de déplacer les curseurs de personnalité (Calme, Extraverti, etc.) vers des valeurs spécifiques, ce qui recalculera et changera instantanément la catégorie de personnalité du Mii.

**Q : Comment obtenir les objets SpotPass qui ne sont plus distribués ?**

R : Dans le **tomodachi life save editor**, il y a généralement un onglet « Special Items » ou « Delivery ». Cocher les cases de ces objets les ajoutera à votre collection comme si vous les aviez reçus via le service de distribution original de Nintendo.
