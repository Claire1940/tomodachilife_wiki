<Section icon={Target} title="Guide d'installation pour PC et console">

L'installation du **mod de performance Tomodachi Life: Living the Dream** dépend de la plateforme choisie. Les utilisateurs de PC utilisant Ryujinx ou Yuzu ont un flux de travail différent de celui des utilisateurs d'une Nintendo Switch modifiée exécutant Atmosphere. Suivez attentivement ces étapes pour vous assurer que les fichiers sont placés dans les répertoires corrects.

<Steps>
  <Step num="1" title="Téléchargez la bonne version">
    Identifiez la version de votre jeu (généralement 1.0.0 ou 1.0.1). Téléchargez le fichier .7z 60 FPS correspondant ou le fichier .zip du mod graphique à partir de sources fiables comme GameBanana ou GBAtemp.
  </Step>
  <Step num="2" title="Configuration du répertoire de l'émulateur (PC)">
    Cliquez avec le bouton droit sur le jeu dans votre émulateur. Sélectionnez "Ouvrir le répertoire des mods". Pour Yuzu, extrayez le dossier ici. Pour Ryujinx, vous devrez peut-être créer un sous-dossier `cheats` et un fichier `.txt` nommé d'après l'ID de build.
  </Step>
  <Step num="3" title="Patching Atmosphere (Switch)">
    Placez les correctifs `.ips` dans le répertoire `sd:/atmosphere/exefs_patches/`. Assurez-vous que le nom du dossier est descriptif (par exemple, `TLLtD-Graphics`) pour garder votre carte SD organisée.
  </Step>
  <Step num="4" title="Activer et vérifier">
    Lancez votre émulateur ou votre console. Dans le menu "Gérer les triches" ou "Gestionnaire de mods", assurez-vous que le mod 60 FPS ou graphique est coché. Redémarrez le jeu pour appliquer les modifications.
  </Step>
</Steps>

<Callout type="success" title="Vérification de l'ID de build">
Pour les utilisateurs de Ryujinx, l'ID de build spécifique pour la triche 60 FPS est souvent `56BF85BD53541346`. Assurez-vous que votre fichier texte correspond exactement à cela, sinon la triche ne se chargera pas.
</Callout>

| Émulateur | Emplacement du mod | Format de fichier | Méthode d'activation |
| :--- | :--- | :--- | :--- |
| **Ryujinx** | `/sdcard/atmosphere/contents/` | .txt (Cheat) | Menu Gérer les triches |
| **Yuzu / Forks** | `/load/[TitleID]/` | Dossier (Mod) | Propriétés > Add-ons |
| **Switch (CFW)** | `/atmosphere/exefs_patches/` | .ips (Patch) | Automatique au démarrage |

</Section>

<Section icon={Shield} title="Optimisation et dépannage">

L'utilisation d'un **mod de performance Tomodachi Life: Living the Dream** peut parfois introduire des bugs visuels, tels que des textures de sol scintillantes ou des icônes Mii floues. Ces problèmes sont souvent liés à la façon dont l'émulateur gère le filtrage anisotropique ou les décalages LOD. Si vous rencontrez une "texture de sol buggée", il s'agit probablement d'un conflit entre le mod graphique et les paramètres de votre pilote GPU.

| Problème | Cause potentielle | Solution recommandée |
| :--- | :--- | :--- |
| **Sol scintillant** | Incompatibilité du pilote | Mettre à jour les pilotes GPU / Passer à Vulkan |
| **Icônes Mii floues** | Mod de décalage LOD | Désactiver le mod graphique / Utiliser l'échelle x1 |
| **Désynchronisation audio** | Désynchronisation 60 FPS | Utiliser un correctif de synchronisation audio dédié |
| **Plantages du jeu** | Charge FPS élevée | Capper les FPS à 45 ou revenir à 30 |

<Callout type="info" title="Astuce de performance">
Dans Ryujinx, l'activation de "Activer l'accès Internet invité" n'est pas liée aux performances, mais s'assurer que "Backend graphique" est défini sur Vulkan peut améliorer considérablement la stabilité des mods 60 FPS sur les matériels AMD et Intel.
</Callout>

| Paramètre graphique | Valeur recommandée | Impact sur les performances |
