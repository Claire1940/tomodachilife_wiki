Bien que le jeu de base ne prenne pas en charge cette fonctionnalité nativement, les joueurs ont découvert des solutions de contournement créatives pour obtenir ce look. Avant de plonger dans les étapes spécifiques, il est important de comprendre les compromis. Parce que la peinture faciale est une texture appliquée sur le visage, elle ne bouge pas avec la logique interne des "yeux" du Mii. Cela signifie que votre Mii aura techniquement un regard "fixe". Cependant, pour de nombreux joueurs, la valeur esthétique d'avoir des yeux bicolores l'emporte sur l'absence d'animation de clignement.

| Caractéristique | Yeux Standard | Hétérochromie par Peinture Faciale |
| :--- | :--- | :--- |
| **Clignement** | Entièrement Animé | Statique / Non mobile |
| **Options de Couleur** | Palette Limitée | Couleurs Personnalisées Infinies |
| **Symétrie** | Symétrie Forcée | Indépendance Complète |
| **Expressions** | Dynamique | Texture Fixe |
| **Facilité d'Utilisation** | Très Élevée | Modérée (Nécessite du Dessin) |

<Section icon={Target} title="Méthode 1 : La Technique de Superposition">

La Technique de Superposition est le moyen le plus rapide d'obtenir l'**hétérochromie dans Tomodachi Life**. Cette méthode consiste à utiliser les yeux Mii existants comme modèle et à simplement peindre une nouvelle couleur sur l'un ou les deux. C'est idéal pour les joueurs qui veulent conserver la forme exacte des yeux Mii originaux sans avoir à les redessiner parfaitement.

Pour commencer, ouvrez l'éditeur de peinture faciale Mii et sélectionnez l'icône de visage souriant située sous l'écran de prévisualisation. Vous devez activer l'option "dessiner par-dessus les traits du visage". Une fois activée, vous pouvez utiliser l'outil pinceau pour colorer l'iris d'un œil.

| Étape | Action | Détail Clé |
| :--- | :--- | :--- |
| **1** | Ouvrir l'Éditeur | Naviguez vers la section Peinture Faciale du Mii. |
| **2** | Activer la Superposition | Cliquez sur l'icône souriante pour permettre de dessiner par-dessus les yeux. |
| **3** | Tracer l'Iris | Utilisez un petit pinceau pour colorer l'iris existant. |
| **4** | Sauvegarder et Tester | Vérifiez le Mii dans l'appartement pour voir l'effet. |

<Callout type="warning" title="Le Bug de Clignement">
Lorsque vous utilisez la Méthode 1, les yeux Mii originaux existent toujours sous votre peinture. Lorsque le Mii cligne des yeux, l'œil peint restera ouvert tandis que la paupière se déplace *derrière* ou *sous* lui, ce qui peut paraître peu naturel en gros plan.
</Callout>

<Section icon={Sparkles} title="Méthode 2 : Dessiner à partir de Zéro">

Pour un résultat plus propre et plus professionnel, de nombreux joueurs préfèrent dessiner les yeux entièrement à partir de zéro. Cette méthode résout le problème visuel du "double œil" où les yeux originaux clignent sous la peinture. En supprimant entièrement les yeux par défaut, vous créez une toile vierge pour votre chef-d'œuvre d'**hétérochromie dans Tomodachi Life**.

<Steps>
  <Step num="1" title="Activer le Dessin de Fonctionnalités">
    Entrez dans l'éditeur de peinture faciale et assurez-vous que le paramètre "dessiner par-dessus les traits du visage" est activé. C'est essentiel pour que la peinture soit visible sur la zone des yeux.
  </Step>
  <Step num="2" title="Traçage Manuel">
    En utilisant les yeux par défaut comme guide fantôme, tracez toute la forme de l'œil, y compris la sclérotique blanche, l'iris et la pupille. Utilisez la fonction de zoom pour une meilleure précision.
  </Step>
  <Step num="3" title="Application de la Couleur">
    Appliquez vos deux couleurs différentes aux iris gauche et droit. Puisque vous dessinez manuellement, vous pouvez même ajouter des dégradés ou des reflets uniques qui ne sont pas possibles dans le jeu de base.
  </Step>
  <Step num="4" title="Supprimer les Yeux par Défaut">
    Quittez l'éditeur et allez à l'onglet standard "Yeux" du Mii. Sélectionnez le type d'œil "Aucun" ou "Masqué". Maintenant, seuls vos yeux hétérochromes dessinés à la main seront visibles.
  </Step>
</Steps>

| Élément | Taille du Pinceau | Stratégie Recommandée |
| :--- | :--- | :--- |
| **Sclère (Blanc)** | Moyen | Remplissez d'abord la forme de base pour couvrir la peau. |
| **Couleur Iris A** | Petit | Dessinez un cercle parfait sur le côté gauche. |
| **Couleur Iris B** | Petit | Dessinez un cercle parfait sur le côté droit. |
| **Pupilles** | Fin | Ajoutez un point noir au centre des deux. |
| **Reflets** | Fin | Ajoutez un petit point blanc pour un aspect "vivant". |

<Callout type="success" title="Meilleur Résultat">
Dessiner à partir de zéro est la méthode préférée pour une création Mii de haute qualité. Cela évite tout problème de chevauchement et permet des formes d'yeux personnalisées que le jeu de base n'offre pas.
</Callout>
