<Section icon={Target} title="Étape par étape : Configurer votre planificateur">

Transférer les données de votre île dans un **planificateur d'île Tomodachi Life** peut sembler être beaucoup de travail, mais le processus est simple. Une fois votre liste initiale téléchargée, la maintenance du tableau ne prend que quelques minutes après chaque session de jeu.

<Steps>
  <Step num="1" title="Initialisez votre île">
    Créez un compte sur le site web de Tomo Board. Vous pouvez configurer plusieurs îles si vous avez différents fichiers de sauvegarde sur plusieurs systèmes 3DS ou émulateurs.
  </Step>
  <Step num="2" title="Importez vos Mii">
    Prenez des photos de vos Mii sur votre 3DS. Téléchargez ces images dans le planificateur et remplissez leurs détails de base : nom, sexe et date de naissance.
  </Step>
  <Step num="3" title="Définissez les personnalités">
    Utilisez les curseurs de personnalité pour faire correspondre vos Mii du jeu. Le planificateur les catégorisera automatiquement en groupes tels que "Indépendant" ou "Confiant".
  </Step>
  <Step num="4" title="Cartographiez vos relations">
    Ouvrez l'outil Carte Sociale. Connectez vos Mii en fonction de leur statut actuel dans le jeu. Vous pouvez utiliser différentes lignes colorées pour représenter les amitiés, les romances ou les rivalités.
  </Step>
  <Step num="5" title="Enregistrez les événements récents">
    Ajoutez des entrées pour tout événement majeur survenu pendant votre session, comme un Mii qui se marie ou un nouveau bébé qui naît.
  </Step>
</Steps>

<Callout type="warning" title="Précision des données">
Vérifiez toujours vos curseurs de personnalité. Une seule différence de cran peut changer la catégorie d'un Mii, ce qui pourrait fausser la précision de votre carte sociale.
</Callout>

| Étape | Temps estimé | Fréquence |
| :--- | :--- | :--- |
| **Configuration initiale** | 30-60 minutes | Une fois par île |
| **Ajout de nouveaux Mii** | 2-3 minutes | Par nouveau résident |
| **Mise à jour du statut** | 1 minute | Quotidien/Hebdomadaire |
| **Examen de la carte sociale** | 5 minutes | Mensuel |

</Section>

<Section icon={Swords} title="Maîtriser la carte sociale et le journal de conflits">

La "Carte Sociale" est sans doute la fonctionnalité la plus puissante du **planificateur d'île Tomodachi Life**. Dans le jeu, vous ne pouvez voir les relations que du point de vue d'un seul Mii à la fois. Le planificateur offre une "vue de Dieu" sur l'ensemble du réseau de connexions de l'île.

Cette vue d'ensemble est essentielle pour gérer les drames inévitables. Lorsque deux Mii se lancent dans un "combat ardent" (les conflits intenses à aura rouge), le jeu ne conserve pas de trace permanente de cet événement. En l'enregistrant dans votre planificateur, vous pouvez construire un historique narratif pour votre île. Peut-être que deux Mii sont "destinés" à être rivaux parce qu'ils se sont battus quatre fois en un mois – c'est une information que seul un planificateur peut vous aider à suivre efficacement.

| Type de relation | Visuel du planificateur | Impact social |
| :--- | :--- | :--- |
| **Meilleurs amis** | Ligne verte épaisse | Forte interaction, faible conflit |
| **Amoureux** | Ligne rose en forme de cœur | Potentiel de mariage/enfants |
| **Rivaux/Ex** | Ligne rouge dentelée | Forte probabilité de requêtes ignorées |
| **Éloignés** | Ligne grise pointillée | Interagissent ou se reconnaissent rarement |

<Callout type="info" title="Conseil de pro pour les rôlistes">
Utilisez la section "Notes" pour chaque Mii afin de suivre leur "lore". Si un Mii demande toujours des vêtements chers ou n'aime que la nourriture épicée, notez-le. Cela ajoute une couche de personnalité qui rend votre île plus vivante.
</Callout>
