## Comprendre la Modification de Sauvegarde
"Tricher" dans Tomodachi Life est essentiellement le processus d'édition de votre fichier de sauvegarde. Contrairement aux jeux traditionnels où vous entrez un code pour déclencher un effet, les éditeurs de sauvegarde interagissent directement avec les données stockées sur votre carte SD. 

| Concept | Définition | Importance |
| :--- | :--- | :--- |
| **Éditeur de sauvegarde** | Logiciel qui lit/écrit les données du jeu | Permet de modifier les objets et les stats des Miis |
| **Custom Firmware** | Logiciel système non officiel | Requis pour accéder aux fichiers de sauvegarde |
| **Checkpoint** | Une application homebrew 3DS | Gère les sauvegardes de vos données de jeu |
| **Sauvegarde NAND** | Une copie des fichiers système | Cruciale pour la récupération après une erreur système |

> **⚠️ Avertissement :** La modification de vos fichiers de jeu comporte des risques inhérents. Créez toujours une copie de sécurité de votre fichier de sauvegarde avant de tenter des modifications. Si vous modifiez incorrectement les relations Mii, vous pourriez subir une corruption du jeu.

*   **Console 3DS/2DS :** Compatible avec tous les modèles.
*   **Carte SD :** Entre 2 Go et 128 Go, formatée correctement.
*   **Ordinateur :** Nécessaire pour transférer les fichiers et exécuter l'éditeur de sauvegarde.

## Utiliser l'éditeur de sauvegarde
Une fois votre 3DS correctement configurée avec le custom firmware et l'application Checkpoint, vous pouvez commencer le processus d'édition de votre jeu. 

1.  Ouvrez **Checkpoint** sur votre 3DS et sélectionnez Tomodachi Life.
2.  Créez une nouvelle sauvegarde (backup) de votre fichier.
3.  Éteignez votre console et insérez la carte SD dans votre ordinateur.
4.  Naviguez vers le répertoire `/3DS/Checkpoint/saves/` pour localiser votre dossier de sauvegarde.
5.  Lancez votre éditeur de sauvegarde préféré (la version 4.0.1 est souvent citée pour sa stabilité) et chargez le fichier `save.dat`.

## Éviter la corruption
L'une des erreurs les plus courantes lors de l'utilisation d'outils fonctionnant comme des **codes de triche Tomodachi Life** est la mauvaise utilisation de la fonction "Friend All" (Tout amies). Cette fonctionnalité échoue souvent à prendre en compte les emplacements de Mii supprimés, entraînant des erreurs de "Mii de Schrödinger" où le jeu tente de référencer des données qui n'existent plus. 

> **💡 Astuce :** Effectuez toujours les modifications manuellement. Lors de l'édition des relations, vérifiez que vous ne jumelez pas des Miis du même sexe de manière à ce que la logique du jeu ne puisse pas le gérer, car c'est une cause fréquente de corruption de sauvegarde.

## Foire Aux Questions

**Q : Existe-t-il de vrais codes de triche par combinaison de touches pour Tomodachi Life ?**

A : Non, le jeu ne contient pas de console de triche intégrée ni de codes par touches. Tous les "cheats" sont mis en œuvre via la modification externe des fichiers de sauvegarde.

**Q : Pourquoi mon éditeur de sauvegarde affiche-t-il une erreur pour mon jeu ?**

A : La raison la plus courante est une incompatibilité de région. La plupart des éditeurs de sauvegarde sont conçus spécifiquement pour les versions US ou EU du jeu. Si vous utilisez une copie japonaise ou d'une autre région, l'éditeur peut ne pas reconnaître le format de votre fichier.

**Q : Puis-je être banni pour avoir utilisé ces outils ?**

A : Tomodachi Life est principalement une expérience solo hors ligne. Cependant, faites toujours preuve de prudence lorsque vous connectez une console modifiée aux services en ligne.

**Q : Que dois-je faire si mon jeu s'arrête de fonctionner après une modification ?**

A : Si vous rencontrez des problèmes, utilisez le dossier "backups" créé par votre éditeur de sauvegarde pour restaurer votre fichier d'origine. Si vous avez suivi le guide correctement, vous devriez avoir une sauvegarde propre disponible pour annuler les changements.
